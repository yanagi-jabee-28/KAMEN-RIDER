# テクノロジーとスーツ仕様書（拡張版）

目的: 美術・SFX・演出・小道具・玩具を横断して使える“実務仕様”。現場での判断基準と安全注意を必ず併記。

---

## コア概念（要点）
- 地脈エネルギー (Geo-Pulse): フォッサマグナ／中央構造線に由来する“地域特性依存”エネルギー。スーツは『有機素材＋ハイテク』のハイブリッドでのみ安定稼働する。
- 伝導原則: 各モジュールは地域素材（エノキ菌糸、そば殻、水引、諏訪精密ギア等）と同調しないと効率が落ちる。

### 共通意匠（制作ルール）
- **胸章ジオメトリ（信濃結）**: 全スーツに共通の幾何学胸章を実装。胸章はLEDレイヤ／光学プリズム構造とし、地域カラーの差分のみを持つ（形状・寸法は統一）。
- **ベゼル＝音とリズムの同一性**: ベゼル回転時のクリック感（3クリック）と5音ジオ・チャイムは全員共通。VFXは基音を共有、地域差は倍音処理で表現。
- **水引シームライン**: 縫い目パターン（幅・位置）は全員共通。素材と色で地域性を出す。これによりカットを跨いだ視認性と継続性が担保される。
- **視覚サインの階層**: 1) 形（共通）、2) 色（地域）、3) テクスチャ（素材由来）—制作は必ずこの優先順位で展開すること。
- **演出ルール**: 変身シーケンス内の"胸章点灯タイミング"と"ベゼル最終クリック"は同一フレーム設計（編集上の同期キー）。
- **玩具適合要件**: 胸章・ベゼル・ジオ・ストーンは玩具での共通インタフェース（LED＋音声トリガー）として定義。寸法公差±1mmで互換性を保つ。

注: 胸章デザインは長野県章の図法に着想を得るが、自治体の公章をそのまま用いる場合は使用規程の確認を行ってください（モチーフ化を基本とする）。

> 制作効用: 共通意匠で視覚的一貫性を維持しつつ、地域差を色・質感で表現することで『同一システムだが地域ごとに固有』という主題を視覚化する.

## 変身・移動・電源（仕様）

### シナノ・コンパス・ブレス（prop/spec）
- 物理: 直径約8cm、厚さ2cm。ベゼル回転機構（4方位クリック）とカートリッジ室（ジオ・ストーン挿入）を備える。  
- 機能モード: Idle / GEO-SET / SYNC (合体増幅) / EMERGENCY（フェイク自爆抑止）。
- UI: 小型OLED風表示（方角・残量・同期インジケータ）。点灯色は担当地域カラーに準拠。
- 実働: フルプロップは安全規格対応の発光ユニット。スタント用は軽量フォーム製で同位置にハーネス接続点を設ける。
- 台本参照値: 変身秒数 8〜12秒（画面上の“合図”でカット合わせする）。

### ジオ・ストーン（in-universe/spec & prop）
- クラス: Region-Seed（N/E/C/S）／Charge-level (0–3)。色と内層模様が地域ごとに異なる。  
- 機能: 郷土愛の情動を結晶化→ブレスと共鳴してスーツへGeo-Pulseを導く。合体時は複数石を同時結合して出力を増幅。  
- 実働指示: 石は中空アクリル＋RGB光学で表現。PR・玩具版は押下で“地域フレーズ”音声を再生。
- 制約: 石のCharge3で個人フルパワー（持続2〜4分）。合体使用で即時に2〜3段階消耗。

### 地脈回廊 (Geo-Line) — 運用ルール
- 出入り口: 道祖神・主要神社・道標に限定。任意地点出力は極めて稀（脚本理由必須）。
- トランジット: 市内短距離は数秒〜十数秒、県横断は数十秒〜数分（作中表現に合わせ調整可）。
- 失敗モード: 出口誤差は“ラストワンマイル問題”を生む（ドラマ化素材）。

## スーツ各モジュール（詳細）

### 神経 — 菌糸体ネットワーク
- 性能: 自己修復（小ダメージで-1秒以内に再伝導）、低レイテンシ（リアルタイム操縦）。
- オペレーション帯域: 最適湿度 60–80%（屋外撮影時はミスト噴霧で再現）。
- 故障モード: 乾燥で伝導遅延（操作遅延 0.5–2s）。修復は湿布・ミスト・短時間休憩で回復。

### 装甲 — ソバロマー・ゲル
- 性能: 衝撃吸収（ダイラタンシー効果）→瞬間硬化（反発係数高め）。
- 視覚: 激突時に“そば湯の香り”演出（現場は匂いを出さず、音/映像で代替）。
- 保守: 乾燥で割れるため、戦闘後は水槽での“水戻し”が必須（プロップは濡れ表現のみ）。

### 動力 — リンゴ酸バイオ燃料電池（カートリッジ）
- 形式: 交換式カートリッジ（小: トレーニング/中: 通常/大: 強化）。
- 運用目安: 中カートリッジで通常戦闘約4–6分、バースト出力で30–60秒の高出力。  
- 安全: 撮影用は発光カプセル＋無害エアロゾル。実機の化学物質は使用不可。

### 筋肉 — SUWA・マイクロマシン
- 性能: 高出力・無音駆動。瞬発トルク高。細かい動作はプログラム依存。
- 故障条件: 砂埃・粉塵で摩耗。現場は防塵キャップとエアブローを必須化。

### 冷却 — A.W.C.S（ワサビ冷却）
- 性能: 短時間で-10〜-30°Cの局所冷却が可能（演出上の数字）。
- 副作用: 催涙性があるため、キャストの直接露出は最小化（撮影はCG/エフェクトで代替）。

### 防御 — 伊那寒天バイオシールド
- 性能: 透過性・耐熱性に優れるが乾燥に弱い。再生に水分供給が必要。
- プロップ: 可動シールドは透明樹脂＋LEDで表現。撮影後は水槽で“復元”カットを撮るとリアリティが上がる。

## 変身・必殺技：演出テンプレ（実務）
- 変身シーケンス（8–12s）: ブレス挿入→ベゼル回転（方角クリック×4）→石点灯（地域カラー）→カメラ引き→スーツ成形（パーツ別ワイプ）→ポーズ。  
- カメラ: 変身ショットは切り返し用に必ず2パターン（ロング／クローズ）。スタントは短めのワープ編集で隠す。
- 必殺技: 発動前に必ず“代償描写”（スタミナ減衰、石の閃光）を1ショット入れること。

## ヒーロー兵装（兵装仕様：美術・アクション・玩具向け）

目的: 各ヒーローの武装を「プロップ仕様・演出・VFX・玩具ギミック・安全指示」で統一する。

フォーマット（各武器）: 名称 / 種類 / in-universe源泉 / prop-spec / VFX / 音響 / 玩具ギミック / 安全注意 / 撮影メモ

### ミズヒキ・ロッド（ホクシン・ブリザード）
- 種類: 捕縛・防御用錫杖型ロッド（多機能ファイバー展開）
- 源泉: 飯田の水引＋光ファイバー（菌糸ネットと同調すると凍結出力を得る）
- prop-spec: 1m前後、収納式ファイバー（リトラクタブルで実線は柔らかいシリコン）、先端に発光ユニット（青白）
- VFX: ファイバー展開→接触で青白の霜エフェクト＋アイシング粒子（画面外の冷却はCGで表現）
- 音響: 細い紙擦れ音＋低周波“凍結チャイム”／捕縛時に短い鐘音
- 玩具: 伸縮ファイバー（安全素材）、LED（凍結エフェクト）、“捕縛”ボタンで音と振動
- 安全: 伸縮ファイバーは柔らか素材で先端にクッションを設ける。実打ちは禁止、接触演出はガイド有り
- 撮影メモ: 捕縛はワイヤー＋後処理の合成で強調。凍結はミスト＋照明で物理感を補完

### サナダ・チューブ・ランサー（トウシン・フレア）
- 種類: 地熱増幅型ランス（プラズマ・貫通アタッカー）
- 源泉: 佐久の地熱・耐熱ガラス＋真空管増幅
- prop-spec: ランス本体1.6m、内部に安全LEDストリップ、先端は発光エフェクト用ソケット（スモークカートリッジ非使用）
- VFX: 発射時に真空管が赤熱→プラズマスパーク（モーションブラー＋焼き跡アニメ）
- 音響: 真空管の低うなり→プラズマ爆発のサブベース
- 玩具: 先端発光＋発射音（非発射プロジェクション）、着脱可能な"六文銭"エフェクトパーツ
- 安全: 高温演出はLED＋煙で代替。投擲演出はダミー使用のみ
- 撮影メモ: 突進カットはワイヤー制御の短距離高速で撮り、衝突はスローとカット編集

### バイオリン・ボウガン（チュウシン・ゲイル）
- 種類: 弦共振式ボウガン（音波弾）
- 源泉: 松本の精密弦楽技術＋共振増幅器
- prop-spec: 高級楓材外装、カーボン弦、音響トランスデューサー内蔵（撮影用は可視弦＋無線発射機構）
- VFX: 弦を弾くと波紋状の“音の刃”が伝播→対象の共振点が赤く点滅→構造崩壊エフェクト
- 音響: 弦の実演音＋ハーモニクスの上乗せ（周波数レイヤーで崩壊感）
- 玩具: 弦を引く感覚と連動するLED/サウンド、セーフティ矢（ソフト素材）
- 安全: 弦テンションは低圧。発射は視覚演出で代替。実弾不可
- 撮影メモ: マクロで弦振動→切替でVFXの“弦が空気を割る”表現を被せる

### オンバシラ・ハンマー（ナンシン・クエイク）
- 種類: 巨大丸太型ハンマー（磁気浮上ギミック付）
- 源泉: 飯田の御柱文化＋リニア超電導技術
- prop-spec: 見た目は数トン級だが実物は軽量フォーム＋中空骨格。浮遊演出用に隠し磁石とワイヤー固定点
- VFX: 打撃時に重力ウェーブ（低周波＋地表クラックアニメ）と粉塵エフェクト
- 音響: 木質の重打音＋地鳴り低音
- 玩具: 可変重量感を模したバネ＆振動ギミック、取り外し可能な“超重磁石”ヘッド（装飾）
- 安全: 大振りはワイヤー＋ダミーで。接触はスタントコーディネーター指示必須
- 撮影メモ: 視覚的インパクトは“振動→地割れ→静止”のテンポで作ると効果的

### プリズム系兵装（湊 つむぎ）
- 種類: 支援用カプセル・指揮バトン・合体中継器
- 源泉: ジオ・ストーン中継回路＋ライチョウ・ローダーインテグレーション
- prop-spec: カプセルは直径15cmの射出モジュール（発光/投下ギミック、内容は医療/冷却/ブースト等）、指揮バトンはコンパス機能内蔵
- VFX: カプセル投下でホログラフィック療法パターン→味方ステータス上昇パーティクル
- 音響: 柔らかいベル＋和声コーラス短句
- 玩具: カプセル射出（安全スプリング）＋ボタンで“地域フレーズ”再生、バトンは振るとライトと簡易合唱音
- 安全: カプセル投下の物理演出はスタントレス（CG合成推奨）。
- 撮影メモ: つむぎの“補給”は必ず市民の反応ショットを挟む（ドラマ重視）

### 兵装共通ルール（制作必読）
- 出力源: 各兵装は基本ジオ・ストーンで補助され、主要消耗はリンゴ酸カートリッジで補填可能（脚本で明示）。
- 非致死性: 設定上、兵装は“局所的・非破壊”が原則。大規模破壊・大量死を示唆する演出はNG。
- クールダウン: 各兵装は連続使用に厳しいクールダウンを持つ（演出で代償を見せること）。

## VFX / SFX 指示（即貼れる箇条）
- 地脈移動: 低周波ドローン（25–80Hz）＋薄い砂埃パルス。映像: 地中ノイズ→亀裂の光学フレア。  
- ジオ・ストーン: 着色光（固有色）＋五度音程のチャイム。合体時は多地域フレーズの重ね録りを用いる。  
- スーツ衝突: ソバロマー硬化は白青のブラー＋“こし”を表す短い低音ノイズ。

## 小道具／衣装管理・安全（オンセット必須）
- 毎ショット前: ジオ・ストーン色/残量インジケータの確認（撮影台帳にUI写真を添付）。
- 防塵手順: SUWA部分は撮影前後に必ずブロー＆カバー。
- 寒天シールド: 濡れ表現後は速やかに乾燥防止処置（専用ケース）を行う。
- スタント: オンバシラ等の大振りはワイヤー＋ダミーで処理。実打ちは禁止。
