# 20 Game Systems and Flow (メカニクスとUX)

本ドキュメントはDesigner/Engineer向けの「プレイヤー体験」定義書である。「なぜこのシステムが存在するか」という設計思想と、プレイヤーが一連の冒険を通じてたどる体験フローを一体で記述する。数値定義は `30_Data_and_Logic_Architecture.md` を参照すること。

---

## 1. コア戦闘思想

### 神の側（敵システム）
- **最適解の提示:** 敵は常に「次ターンで自軍被害を最小化しプレイヤーを確実に削るための行動」を、**滲みや揺らぎの一切ない端正なUI**で予告する（エントロピーの凍結）。
- **学習と適応:** 物語の進行に伴い、神はプレイヤーの「自傷による回避パターン」すら学習・予測し始める。「自傷すれば勝てる」とルーチン化した瞬間、神は自傷を前提とした罠を張る。摩擦は常に「賭け」でなければならない。

### 人の側（プレイヤー）と三条の熱源
- **意図的な非合理（摩擦行動）:** プレイヤーは「完璧な対抗手段」ではなく、**「神が計算困難な非合理な熱量」**を用いて神の予告UIを物理的に破壊し、演算エラーを生じさせる。これを以下の3つのリソースで管理する。
  1. **活魂（Kakkon）**: 肉体の生命力と持久力の統合。「生存そのもの」を指す単一のゲージ。0になれば泥に還り（戦闘不能）、尽きかければ神のクリティカル率が100%になる「無防備」状態へ陥る。
  2. **情念（Jonetsu）**: 怒りや悲しみのバグ燃料。被ダメージや仲間の危機で蓄積し、神のUIを破壊する技のコストとなる。
  3. **摩擦熱（Friction）**: 武具の連続使用による負荷。威力は増すが閾値を超えると「過熱（沈黙）」し、一時的に使用不能になる。
- **予測の限界突破:** 神が学習し予定調和な行動が通じなくなった後半では、「自律NPCとしての行者うかみの乱入」や蓄積した情念を一気に解放する極大代受苦だけが高度化した計算を粉砕する鍵となる。

## 2. 装備スロットの運用論理（物理的制約と摩擦）

「武器は震える腕を支える杖であり、傷を隠す盾である」という地の底を這う人間の哲学を、4スロットの運用制約として定義する。

| スロット | 名称 | 定義 |
|---|---|---|
| 1 | **主腕（Main Arm）** | 物理破壊・直接攻撃を担うメインスロット。1H/2Hの判定を持つ。 |
| 2 | **副腕（Sub Arm）** | 予測UI攪乱、回避、補助を担う「ノイズ具」スロット。 |
| 3 | **装束（Shozoku）** | 防御衣服。ダメージ履歴が金継ぎの傷跡となり、防御力(PTG)を形成する。 |
| 4 | **形代（Katashiro）** | 最大2個。アクセサリではなく「遺品・祈り・未練」を装備する枠。 |

### 腕の運用論理（相克と摩擦）
* **両手占有（2H）の掟:**
  大剣、槍、弓といった質量の大きい武具は「主腕」に装備した時点で「副腕」の使用を封じる。両手で泥を掴まねば、天を揺らす熱量は生まれない。
* **片手運用（1H）と副腕の相克:**
  短刀や扇などの片手武具は「副腕」との併用が可能だが、非訓練者（ミコト、スクナ等）がこれを行う際には**「構えの摩擦（Stance Friction）」**が伴う。防御と攻撃の予備動作が神の予測アルゴリズムにわずかな「隙（バグ）」を与える代償として、**武器摩耗係数 k は1.2倍**へと跳ね上がる。演出上では足元が泥に滑るような不格好なモーションとなるが、ミコトはこの摩擦（ノイズ）を逆に情熱へ変え、**「戦闘中の情念ゲージの上昇率も1.2倍になる」**というパッシブスキル（情念の昇華）へと昇華させる。不完全だからこそ熱量が生まれるという本作のコア・テーマのメカニクス化。
* **固定裝具（Fixed Gear）:**
  「手甲（BEAT GAUNTLET）」やスクナの「石乳鉢」のように、魂と一体化した装具はスロットを消費せず、装備に関わらず常に機能する。

## 3. 神写し（技の継承）

- **習得方式:** 仲間が特技を使用するたびにミコトの「理解度ゲージ」が蓄積し、閾値で習得する。単なる回数稼ぎを防ぐため、**瀕死・属性不利など「苦境」での使用時に理解度が爆発的に上昇する（苦境こそが継承の触媒）。**
- **共鳴（ユニゾン）と戦術的シナジー:** 武具の「持ち方」を共有し、魂の欠損を補い合う瞬間、理を砕く共鳴が生まれる。
  - **大気拳・泥の瞬き（ミコト×うかみ×タチバナ）**: ミコトが槍に持ち替えたターンに、うかみが石鉾、タチバナが銛を構える。3者の「槍」が共鳴し、神の防御UI展開の0.1秒前に理を爆砕する。
  - **星砕きの槌音（マヒト×スクナ）**: マヒトの大槌の打撃に、スクナの「粉砕・重杵」が重なる。装甲の亀裂に劇薬が浸透し理を内部から腐食させる。
  - **産霊の咆哮（うかみ×ウズ）**: 行者うかみが法螺貝を吹き、ウズが旗槍による攪乱の舞を披露。戦場全体にノイズの嵐が発生し敵UIを「判読不能」にする。
- **うかみとの特殊共鳴:** 第4幕で、ミコトが継承した「うかみの技」と、自律NPC化した行者うかみの行動が同一ターンに偶然重なった瞬間、**確定で神の理を粉砕する絶大な共鳴効果が発動**する。

### うかみ離脱・帰還の戦術変化
| 第1〜2幕 | 通常参戦 | 最強の庇護者として依存感を植え付ける。序盤はうかみがいれば安定。 |
| 第3幕 | **一次離脱** | 絶望的な戦力低下。ミコトのうかみ継承技だけが生命線となる。 |
| 第4幕 | **自律NPC化** | 黄泉領域限定で「独立リソースAI」として参戦。技の使用や熱量はうかみ自身のAI内で管理（画面共有リソースを消費しない）。「情念」や「肩代わり」の演出により、隣で戦ってくれている頼もしさを強調する。 |
| クリア後 | **通常復帰** | 制限なしの正規4名枠に復帰。 |

- **行者の誓約（空間的断絶）:** 第4幕のうかみは「行者の誓約」により別天津神撃破まで、黄泉・常世の領域にのみ拘束される。地上にいる間の遠隔守護といった支援は**一切なし**。空間的な断絶を明確にすることで、地上のミコトたちの自立と、黄泉での再会のカタルシスを際立たせる。
- **第3幕の「名もなき影」:** 第3幕特定イベントのみ、一度限りで謎の「影」がミコトを庇って消滅する。これはカガセオの意識の残滓（星の瞬き）であり、うかみの通常システムとは別の単発イベントスクリプトとして処理する。

## 4. 武器修復サイクル：金継ぎ・代受苦・呼び継ぎ

「使用→過熱（代受苦）→修復（金継ぎ）」の循環が、本作の主題「泥臭い修復と継承」そのもの。

### A. 金継ぎ（歴史の受容）
- **概要:** 耐久が尽きる前に修復し、使用・被ダメージ履歴を特効・耐性・付与特性へ変換する。
- **材料:** 純粋な「星の砂（カガセオの破片）」のみ。
- **結和神への道:** 同一武器に星の砂のみで丁寧な金継ぎを繰り返すと、愛着の履歴密度が閾値を超え「結和神（自律する相棒）」へと覚醒する。

### B. 代受苦（過熱・沈黙循環）
- **概要:** 任意のタイミングで発動可能。残存耐久値をすべて「薪」として焼却し、莫大な熱量の一撃を放つ。発動後は武器が「過熱・沈黙状態」となり一時使用不能になる。
- **耐久度ゼロ時の挙動:** 代受苦を使わずに耐久度が0になった場合、武器は強制的に「過熱・沈黙」状態へ移行する。この場合、大ダメージを与える代受苦の恩恵は一切発生せず、ただ武器が使用不能になる。これにより、耐久が尽きる前に修復するか、直前に代受苦を叩き込むかというリソース管理のチキンレースを強いる。
- **発動威力:** 残耐久が低いほどオーバーフロー計算が働き破壊的な威力になる（チキンレース）。金継ぎ・使用の修復回数も威力係数に含まれるため、「大切に使い続けた歴史」が強さとなる。
- **極大代受苦（Extreme Daijuku）:** 通常は過熱・沈黙の循環止まりだが、特定のボス戦（ツクヨミ・別天津神等）限定で「武器の完全消滅」を選択できるモードがアンロックされる。発動時、武器は「遺骨（Remnant_Bone）」を残して永遠に消滅する。

### C. 呼び継ぎ（キメラ接合）
- **材料:** 極大代受苦で生まれた「遺骨」のみに使用を限定する。通常の星の砂では意図的キメラ化は起きない（事故防止）。
- **効果:** ベース武器に遺骨を接合し、失われた武器の特性（特効・スキル）を移植する。
- **代償:** 摩擦係数（Friction）が極端に上昇し、装備名が `天叢雲...（鉄パイプ）...剣` のようにノイズ混入した明滅表示になる。
- **魂の摩耗（Soul Attrition）:** キメラ化済みの武器に再度ボス級の極限代受苦を行っても遺骨は生成されず、完全に塵となって消滅する。真の別れとなる。

### D. 秘奥義
| 技名 | 効果 | 代償 |
|---|---|---|
| **逆流・代受身** | 武器耐久を消費する代わりに使用者の活魂最大値を戦闘中に「1」にする。行動最速化・確定クリティカルを得る。 | 活魂1の急激なリスク |
| **剣塚の共鳴** | 過去に代受苦で折った全武器の幻影を召喚し一斉攻撃。威力はゲーム通算の代受苦回数・履歴密度に比例して青天井に伸びる。 | ゲーム後半ほど強力になるが、消費が重い |

## 5. 黄泉戸喫（不可逆のシステム変化）

黄泉比良坂探索中、プレイヤーは「黄泉の食べ物」を食べるかどうかを問われる。

**摂取した場合：`Yomotsu_Eat_State` が発生し、以下の変化が永続する：**
- 通常の回復アイテム・スキルがすべて**毒ダメージ**に反転する。
- 自傷行動や敵から受けるノイズが**活魂吸収**に変化し、「侵食ゲージ」が蓄積する。
- 侵食ゲージ超過で毎ターン侵食ダメージと最大活魂低下が発生する。
- **地上へ帰還すると通常回復が再び有効になるが、UIの活魂ゲージには黒墨と金継ぎラインが永続表示され、「二度と元に戻れない」ことを視覚的に刻む。**

**ロール別対応チャート：**

| キャラ | 黄泉戸喫状態での役割 |
|---|---|
| スクナ | 黄泉対応型劇薬を調合し、侵食進行を唯一遅延させる |
| うかみ（行者） | 法螺貝＋錫杖の範囲祈祷で侵食ダメージを一時停止する結界を張る |
| タチバナ | 身代わり呪術で侵食を自身に引き受ける |
| マヒト | 戦闘中に即席金継ぎを行い、黄泉泥で脆くなった武器を延命させる |
| ウズ / ワカヒコ | 黄泉の泥を攻撃触媒化し、敵UIに致命的なバグを引き起こす |

## 6. ワールド導線と主要エリア

- **基本構造:** 周縁集落を巡礼して中心領域（ヤマト）へ迫る円環導線。
- **仲間合流エリア一覧:**

| エリア | 仲間 | チョークポイント（関所）と引力の必然性 |
|---|---|---|
| アマ（漁村）「星落ちの磯」 | **うかみ** | **最初の摩擦**: 漂着したミコトというノイズに惹かれ、彼に「最初の熱量（赤いスカーフ）」を巻きつける。 |
| 泥の郷「澱んだ祈祷所」 | **スクナ** | **理不尽な死の壁の迂回**: 猛毒地形による進行不可を突破するため、猛毒という「科学的ノイズ」を求めて祈祷所へ寄り道する。 |
| 灼熱の底「灼熱たたら場」 | **マヒト** | **目的への渇望**: 武具の限界（過熱）を手当てするため、神の威を恐れぬ異端の鍛冶師を求めて自ら探訪する生存の限界的必然。 |
| 天の関所「白堊の回廊」 | **ウズ** | **絶対のチョークポイント**: ヤマトへ至る直線の防壁。ミコトの接近によるシステムの「隙」が、窒息しかけていた彼女の狂騒を内側から爆発させる。 |
| 暗礁地「忘却の海食洞」 | **タチバナ** | **海路迂回の障害**: 街道完全封鎖による海路の極限迂回。過去の生贄の怨念による道塞ぎに対し、その執着を泥水まみれで受容し道を開く。 |
| 絶壁の空「天望の天守」 | **ワカヒコ** | **山岳監視砦の超克**: 険しい山岳迂回ルート。空から防衛網を敷く冷淡な狙撃手に対し、地を這う泥臭さを見せつけることで彼を空から引き摺り下ろす。 |

- **葛城山:** うかみ一次離脱の確定舞台。天の包囲網でうかみが自らを犠牲に退路を確保する。離脱時、自身の「手甲（BEAT GAUNTLET）」をミコトに託すと同時に、神写しフラグとして**「うかみの基礎防衛技（法螺貝等による挑発・防御の代替技）の理解度を強制的に100%にする」**処理を行い、習得させることで第3幕の詰みを防止する。

## 7. 星土の脈継ぎ（ファストトラベル）

- **概念:** 複雑なパズルを排除し、「祈り」という直感的なアクションに純化したファストトラベルシステム。
- **操作フロー:**
  1. フィールドで未開通の「祠（ほこら）」を発見する。
  2. 近づいて「祈る」を実行すると、短い祈り演出の後、世界マップ上でその祠から最寄りのネットワークへ黄金の線（金継ぎ）が自動的に走る。
  3. 以降、その祠は永続的なワープポイントとなる。
- **神社建立:** ボス撃破で得た「御霊」を拠点に「祀る」だけで大型ハブ「神社」が建立される。神社ではアマテラス再起動儀式や特別クラフトが可能になる。
- **神器との連動:**
  - **八尺瓊勾玉:** 神社ネットワーク全体の「受容力（接続社数×祈りの総量）」を結晶化して作成する。
  - **八咫鏡:** ネットワークの遣い回しではなく、**ミコトの「赤いスカーフ」とゲームプレイ全体の「金継ぎ回数データ」**を参照して作成する。金継ぎを多く重ねた歴史そのものが、神を再起動させる鍵となる。

## 8. 敵生態系と戦闘挙動

| 系統 | 特性 | 戦闘的役割 |
|---|---|---|
| **白化神** | 予告精度最高、学習速度速 | 定型戦法・摩擦行動のルーティン化をカウンターする |
| **澱神** | 予告UIが乱れるバグ型 | 読めない行動で事故を誘発する |
| **荒魂獣** | 直前変動行動・野生型 | 即応判断を要求する。白化への防衛反応で凶暴化した野生生物 |
| **棄物** | 修復されず遺棄された器物の怨念。破損履歴デバフ | 悲しみの攻撃で「修理し続けること」の大切さを問わせる |
| **擬神兵** | 不完全な神の模倣。高精度だが崩壊する隙を持つ | 人間の不完全性を逆用できる教えを込めた敵 |

**主要ボスの特殊ギミック：**
- **ツクヨミ（偽ラスボス/アンチウイルス）:** 「食事(回復)」「代謝アクション」を穢れとして検知するAIを持つ。プレイヤーが最も穢らわしい行動（泥団子投げ、ゲテモノを食う等）を取るとツクヨミの計算UIが嫌悪・処理不能でエラーとなり特大の隙が生まれる。
- **アメノイワトワケノカミ（天岩戸）:** ダメージゼロの物理破壊不可ボス。神社ネットワークの過負荷を流し込み、狂騒のノイズで亀裂を入れることで進行する。
- **黄泉軍勢:** 行者うかみ合流前は全員ダメージ無効・予告完全ブロック。うかみの祈祷バフが及ぶ範囲でのみ無効化が解除され撃破可能となる。

## 9. UX体験曲線とUI仕様

### ゲーム体験の感情設計

| フェーズ | UX意図 |
|---|---|
| 第1〜2幕 | うかみへの依存感を徹底的に育てる。彼がいれば何とかなる安心感。 |
| 葛城山（離脱直後） | 画面の操作可能キャラが3人に減り「穴」が視覚的に生まれる。神写しの「うかみ技」が唯一の希望に。 |
| 第3幕（偽終幕直後） | ツクヨミ勝利の高揚を一瞬で白一色の凍結UIに塗り替え、絶対零度の静寂が世界を覆う演出を挿入。 |
| 第4幕（うかみ帰還） | 5人目アイコンが不規則なノイズと共にUIに飛び込み、タイムラインが5列に広がる圧倒のスペクタクル。 |

### 主要UI仕様

- **タイムラインUI:** 神の予告が端正な筆致で明示される。ノイズ発生時、UIの線に物理的な亀裂が入り絵の具が滲み出す。破壊時はBGMの逆再生・文字化けなど「神のUI自体がバグる」演出を実装する。
- **地図UI:** ひび割れた和紙を模したデザイン。祠発見時に黄金の線が走る演出が入る。
- **装束スロットUI:** ダメージ履歴に応じてUIの衣服アイコンが物理的に破れ、修復によって金継ぎの線が増える。防御力の蓄積を視覚で直接確認できる。
- **呼び継ぎUI:** キメラ武器は武器名がノイズ混入状態で明滅表示される（例: `天叢雲...（鉄パイプ）...剣`）。
- **剣塚の共鳴UI:** 発動時、画面背後にこれまで代受苦で消えた全武器名が高速ログとして流れ落ちる。
- **星継ぎの系譜UI（ポジティブな墓標）:** 代受苦・呼び継ぎで失われた武器の名前と修復履歴を讃える正の記録リスト。「武具の墓標（廃案）」ではなく継承の軌跡として機能する。

## 10. クリア後コンテンツ（傷跡の海）

- **行者還しの儀:** うかみを地上へ取り戻す3フェーズ特殊儀式。障壁ボス「八雷神（ヤクサノイカヅチ）」との死闘。これは「死者蘇生魔法」ではなく、ミコトの手甲という強烈な魂の繋がりとパーティの熱量によってのみ成立する唯一のバグ（特例）である。

- **幻曜の代受苦:** 傷跡の海で取れるサルベージ素材で、過熱した武器を即座に再冷却・再使用するシステム。「毎ターン代受苦→即泥で再冷却→また代受苦」の無限ループを解禁する。
