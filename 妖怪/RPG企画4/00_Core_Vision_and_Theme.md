# 00 Core Vision and Theme (思想と基盤)

本作の開発におけるすべての仕様、美術、物語、数値調整の善悪を判定するための「不変の憲法（マスターピッチ）」である。開発メンバーは迷いが生じた際、必ず本ドキュメントの原則に立ち返ること。

## 1. プロジェクト基本情報 (基礎情報)
- **プロジェクトコード:** RPG企画4
- **タイトル:** アシブネノミコト 〜天降（あまくだ）る御子と、星屑の大地〜
- **ジャンル:** 和風神話ダークファンタジーRPG
- **コア・エクスペリエンス:** 「泥臭い修復と継承」を体感する、痛みを伴うエモーショナルな旅。

## 2. コア・テーマと基盤哲学 (前提知識)

**「悲哀による永遠の結晶化（停滞）」と「傷を抱えた命の還流（変転）」の衝突。**

本作の世界は、二つの相反する「慈悲」の衝突で構成される。
愛する対象（イザナミ）の腐敗と喪失という痛切な悲哀を恐れ、悲劇を二度と起こさないよう世界を美しい「不変の結晶（琥珀）」に凍らせて保存しようとする天の神々（イザナギの理）と、死や別れの悲しみを受け入れ、壊れたものを修継ぎ（金継ぎ）しながら、それでもなお命を燃やし、魂を次代へ還流（めぐ）らせようとする地上の者たちの物語である。

### 主人公ミコトの本性：結晶世界の「要石（かなめいし）」
主人公ミコトは、本来「天孫降臨」におけるニニギノミコトを原型とし、地上（国譲り）を平定し、不変の結晶世界の「要石（コア）」となるべく遣わされた「空（から）の器」である。しかし、地上で人間の熱に触れ、反逆の星神カガセオの破片を宿したことで、天の計算を狂わせる最大のノイズとなる。
序盤では「天の使者としての役割」を自覚せず漂着するが、大地の熱に触れることで「ただの人間」へと脱皮していく過程を描く。

### 結末の哲学：永遠の拒絶（イワナガとサクヤの脱構築）
本作は日本神話の「コノハナサクヤヒメ（寿命と変化）」と「イワナガヒメ（不変と永遠）」の逸話を脱構築する。
主人公は、物語の終端で神の血脈へ連なる特権（＝永遠の静止）を否定する。悲しみを無くすためにすべてを止める「永遠（イワナガ的な呪い）」を打ち砕き、朽ちゆく寿命と劣化（サクヤ的な生）を選び取った「歴史上最初の『ただの人間』」として着地する。
クローズドで完璧なシステムに、「寿命（劣化と死）」という不可逆のバグを実装する行為こそが、神話からの独立である。

## 3. ゲームデザイン原則 (深い洞察)

すべてのゲームメカニクス、UI、レベルデザインは以下の原則に従い構成されなければならない。
**「ゲームのあらゆる仕様は、効率的な消費ではなく、泥臭い修復と継承をプレイヤーに体感させるためにのみ存在する。」**

### 3.1 神の振る舞い（敵・システム側の作法）
- **静謐とエントロピーの凍結:** 神々は自己の傷を極端に恐れ、世界を完全に予測・管理しようとする。
- **最適解の押し付け:** 戦闘時、神は常に「完璧に計算された予告UI」を展開し、プレイヤーを理の中に閉じ込めようとする。彼らの行動に無駄はなく、冷酷なまでに理にかなっている。

### 3.2 人の振る舞い（味方・プレイヤー側の作法）
- **ノイズと摩擦による反逆:** プレイヤーは神の計算を上回るのではなく、神が理解できない「非合理な熱量（自傷、過熱、代受苦など）」を用いることで予測計算に強烈なノイズ（エラー）を発生させる。
- **神を圧倒する狂気（死狂い）:** あえて死の淵（活魂ゼロ・情念ありの狂気）に飛び込むことで、神の最適予測UIを物理的に爆砕する。死んでいるはずの者が動くという「計算不能なノイズ」を見せつける戦術の極地。
- **傷の肯定（金継ぎ）:** 傷つくことを恐れないのではなく、傷ついた器を「星の砂（金継ぎの光）」で美しく修復し、その修復の歴史（Friction）自体を強さの源泉とする。
- **付喪神顕現と必殺連動:** 金継ぎ履歴が蓄積して意志を持った武具（付喪神）にのみ、理を砕く最終手段「極大代受苦」が許される。必殺は消費の延長ではなく、相棒の存在証明と引き換えに世界の結晶化へ亀裂を入れる契約行為である。
- **白堊の回廊の絶望:** 第2幕中盤、ヤマトへ至る唯一の道「白堊の回廊」で白化神・守護兵の完璧な見切りが連続し、いかなる美しい武器も読まれて無効化される。この区間は意図的な過負荷として設計され、壁は一度突破できるもののプレイヤーに強烈な敗北感と“宿題”だけを残す。ウズの狂騒による内側からの爆砕が発動するまで、既存の手段はすべて無効化される。ここでの経験は、単なる敵強化ではなくシステムの前提を壊す体験であり、マヒトの禁忌鍛造＝レベル2への導線をプレイヤーの胸に叩きつける。
- **段階的鍛造解放（不便から超越へ）:** 武具メンテナンスは「どこでも同じことができる便利機能」にしない。加入前は野営地の応急処置（Lv0）と設備拠点での金継ぎ（Lv1）に限定し、マヒト加入後に前線でのLv1運用が解禁される。さらに、加入直後は拠点限定で禁忌鍛造（Lv2）を解放し、物語イベントを越えた後にのみ野外Lv2を許可する。最終的な神器錬成（Lv3）は大型神社と第4幕条件を満たしたときだけ開く.
- **物理現象の神話的翻訳:** SF的なガジェットやシステム用語（アース、ショート、超新星残骸など）をそのまま用いるのではなく、「純粋な物理現象（熱量、残留応力の解放、摩擦、接地の放電、星の瞬き）」として神話世界へ完全に翻訳・統合する。人間の泥臭い抵抗は、常に極限の熱量と物理的な事象（泥の絶縁盾、流星の熱破壊）を伴って神の「理（システム）」に逆らう。
- **神話的特異点の許容:** 本作が拒否するのは「説明責任を放棄した便宜的な便利魔法」であり、祈り・執念・誓約が自然法則を歪める極限現象そのものは公式に許容する。スクナの劇薬、ウズの舞、天岩戸開きの狂騒は、理の外に逃げる超常ではなく、理に亀裂を入れる神話的実在として扱う。
- **流体力学的なノイズの共鳴（引力と摩擦のシナリオ構造）:** 完璧な理が敷かれた世界を整然と流れる「層流」とするならば、ミコトという強烈なノイズがそれを逆行していく過程で、システム内部に潜んでいた「バグ（各キャラクターの狂騒や執着）」が連鎖的に誘爆し、ミコトのもとへ吸い寄せられていく。キャラクターとの出会いは偶然ではなく、理の壁（チョークポイント）を突破する必然と、ノイズの摩擦が引き起こす物理的な引力によって成される。

### 3.3 外部参照要素の翻訳原則（DQ要素の昇華）
- **直輸入の禁止:** 外部RPGの記号（メタル系、封印、反射、テンションなど）を名称や数値だけで移植しない。必ず「結晶化 vs 還流」の対立軸へ翻訳してから導入する。
- **本作での採用翻訳:**
	- 「メタル系」→ **剥落の星屑**（高防御・短時間帰還・高報酬）
	- 「特技封じ」→ **凍結の真空**（情念依存技を封じる静止）
	- 「場ルール改変」→ **無菌の帳 / 血の泥沼**（結晶化側と還流側の環境圧）
	- 「テンション」→ **共鳴過熱**（新ゲージではなく、情念上限突破と耐久焼却で表現）
	- 「反射」→ **理の反射鏡**（八咫鏡を介した限定反射）
- **UI肥大の禁止:** 新規ゲージを増やして解決しない。活魂・情念・武器耐久度の三条熱源の内部結合で表現する。
- **神話翻訳の徹底:** 現代機械語ではなく、和語と物理現象で意味づける。たとえば「封印」は「静止の吸納」、「反射」は「理の逆転」として記述する。

## 4. ビジュアル・フィロソフィー（同一キャンバス原則）

この世界は、「一枚の生きた和紙（Washi）」の上に、「岩絵具（Iwa-enogu）」이라는同一の画材で描かれる。神と人で3Dやセルルック等の「素材の断絶」を起こしてはならない。両者の違いは、画材ではなく**「筆致（マチエール）」**のみで表現される。

- **天の理（神）:** 滲みを最小限に抑えた、端正で冷酷な細線と制御。凍結した格調の高さ。
- **地の泥（人）:** 和紙の線を引っ掻くような掠れ（Kasure）と、岩絵具の奔放で不規則な滲み（ノイズ）。土と血の匂い。

これらの世界観を補強する具体的な設定、システム、データ要件、および美術の生成ルールについては、以降のドキュメント（10〜40）にて詳細に定義する。
