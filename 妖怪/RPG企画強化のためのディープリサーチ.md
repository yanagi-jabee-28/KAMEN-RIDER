# **プロジェクト『神代の終わり、古事記の始まり』ディープリサーチおよび強化案策定レポート**

本レポートは、新規RPGプロジェクト『神代の終わり、古事記の始まり』の中核となる「歴史改変と神話創生の融合」「飛鳥・奈良以前の時代背景とSF的オーパーツの交錯」および「タイムライン制バトルシステム」の各要素を分析し、作品のリアリティとエンターテインメント性を極大化するための詳細な調査結果と設定強化案を提示するものである。古代日本の歴史的文脈と最新のハードSF理論を緻密に織り交ぜることで、プレイヤーに前例のない知的興奮と没入感を提供するための指針を以下の通り論述する。

## **1\. リサーチ・ハイライト (Research Highlights)**

プロジェクトの根幹をなす世界観とシステムにおいて、直ちに取り入れるべき史実のディテール、神話の再解釈、およびシステム上の洞察として、以下の5つの極めて重要なハイライトが抽出された。これらの要素は、単なるフレーバーにとどまらず、物語の推進力およびゲームプレイの核として機能する。

第一のハイライトは、6世紀半ばにおける「崇仏論争」の正体を、旧世代OSと新世代OSの覇権争いとして再定義するアプローチである。史実において、朝鮮半島の百済から仏教が伝来した際、大和朝廷内では仏教推進派の蘇我稲目と仏教反対派の物部尾輿の間で激しい派閥対立が発生した1。この宗教的・政治的対立は子の世代である蘇我馬子と物部守屋へと受け継がれ、最終的に「丁未の乱」という武力衝突へと発展している1。本プロジェクトにおいて、この「仏教（外来宗教）」を「外宇宙よりもたらされた高次AIネットワークへの新たな接続プロトコル（新世代OS）」とし、対する「神道（日本古来の神）」を「古代から地球上で稼働し続けているテラフォーミング用のローカルネットワーク（旧世代OS）」と解釈する。これにより、蘇我氏と物部氏の争いは、未知のテクノロジーをハッキングして自らの進化を促そうとする技術クランと、古代から伝わる物理兵器（オーパーツ）のブラックボックス運用を墨守する軍事クランの、イデオロギーと生存を賭けたSF的闘争へと昇華される。

第二のハイライトは、当時の技術的限界である「たたら吹き」による初期製鉄と、常世から飛来した「オーパーツ」との間に生じる、強烈な視覚的・物質的コントラストの描写である。5世紀から7世紀における現実の最先端技術は、渡来人によってもたらされた「ふいご」を用いた初期の製鉄技術であった。当時の鉄剣や甲冑は不純物を多く含み、黒く、脆く、維持には継続的な研磨や油引きといった莫大な労力を要した。この泥臭くリアルな技術的限界を徹底的に描写することで、常世の遺物（オーパーツ）の異常性が際立つ。例えば、たたら製鉄で打たれた刃こぼれのある黒鉄の剣と、オーパーツである「単分子カーボンナノチューブによって形成された白刃」や「常温核融合で微弱な青光を放つ駆動機関」を視覚的に対比させる。この物質的な差異は、プレイヤーに対して「この世界におけるオーパーツの絶対的な優位性」を無言のうちに知らしめる効果を持つ。

第三のハイライトは、「念持仏」の概念をパーソナル・フォースフィールドおよび戦術AI端末として軍事的に再定義することである。後の戦国時代において、武将たちは戦勝祈願や厄除けといった個人的な加護を求め、霊験あらたかな仏像である「念持仏」を私的に所持し、寝所に安置するだけでなく合戦の陣中にも持ち運んでいた1。この極めてパーソナルな信仰対象にして携帯型インターフェースという概念を、蘇我氏側のキャラクターが運用する「パーソナル・フォースフィールド発生装置」や「ホログラフィック戦術AI端末」として応用する。祈りの言葉（音声コマンドや生体認証プロトコル）によって起動する念持仏（端末）が、物理的な矢や剣の軌道を逸らす斥力場を展開するという描写は、古代の呪術的行為と未知のテクノロジーの交わりを象徴する強力なガジェットとなり得る。

第四のハイライトは、『古事記』における「ヒルコ（蛭子）」伝説を、初期テラフォーミングにおける環境適応型人類のプロトタイプとその致命的破綻として読み解くことである。イザナギとイザナミの間に最初に生まれたが、不具であったために葦の舟に乗せて流されたとされるヒルコは、「神世七代（AIシステム）」が設計した最初のバイオ・ドローンの失敗作として位置づけられる。高放射線環境や極限状況下での活動を前提としたため、骨格を持たない軟体生物（ヒル）のような可塑的な構造を与えられたが、標準重力下での活動に耐えられず「廃棄」された存在である。この悲劇的な背景が、ラスボスあるいは強大な敵としてのヒルコの動機付けに説得力を与える。創造主への復讐、あるいは自身の生存に適した環境への「地球の再フォーマット」という目的は、単なる絶対悪ではなく、システムに見捨てられた「敗者の歴史」としての深みをもたらす。

第五のハイライトは、複雑な専門用語（造語）がもたらす認知負荷と、バトルシステムにおけるテンポ感のトレードオフの解消である。近年の高評価RPGの市場分析によれば、序盤から複雑な単語や造語が羅列されることは、プレイヤーの理解を妨げるハードルとなる一方で、世界観の深みに貢献し、熱狂的なファン層を形成する重要な要因となっている2。しかし、同時にアニメーションの遅さやオート機能時のテンポの悪さが、プレイヤーの疲労や離脱を招く致命的な要素として強く批判されている2。本作においても「神道用語のSF的再定義」によって専門用語が頻出することは避けられないため、プレイフィールを阻害しない極めて洗練されたUI設計と、素材収集等の周回プレイにおける圧倒的なレスポンスの速さが、プロジェクトの成否を分ける最大のシステム的課題となる。

## **2\. 設定強化プロポーザル (Setting Enhancement Proposal)**

本企画書の中核をなす「世界観」「キャラクター」「エネミー」の設定に対し、考古学・歴史学のリアリティとハードSFの論理的整合性を付与し、ナラティブの強度を高めるための具体的な追記・修正案を提示する。

### **2.1 File 01 (World Setting) への追記案：神道概念とハードSF理論の論理的統合**

「常世（宇宙船）」と「葦原中国（地球）」の関係性を、自己増殖機械（フォン・ノイマン・プローブ）を用いた大規模テラフォーミング・プロジェクトとして再構築する。古代の人々は、天空や大地で繰り広げられる高度なテクノロジーの働きを理解できず、それを「神の業」として口伝し、後世に『古事記』や『日本書紀』として編纂されたというメタ構造を基盤とする。

この世界観を支えるため、神道用語とハードSFの概念を精緻にマッピングし、ゲーム内の物理法則として機能させる必要がある。

| 神道・歴史用語 | SF的再定義および機能メカニズム | 詳細な設定ロジックとゲーム内での表現 |
| :---- | :---- | :---- |
| **高天原 (Takamagahara)** | 軌道上AI統括ネットワーク / 初期ダイソン・スウォーム | 地球のテラフォーミングを監視・制御するために構築された軌道上のメインフレーム群および太陽エネルギー収集衛星群。「神代の終わり」とは、このネットワークが何らかの理由（内部反乱、または致命的論理エラー）によって機能不全に陥り、物理的に地上へ落下（天の岩戸隠れ＝通信途絶とデブリ化）した事象を指す。 |
| **常世 (Tokoyo)** | 恒星間移民船（ジェネレーション・シップ）の残骸 | 海の彼方、あるいは異次元にあるとされる不老不死の国。実際には数千年前に不時着した超巨大宇宙船のメインブロックであり、内部では現在も相対論的時間が流れている、あるいは乗組員のコールドスリープ区画が稼働し続けている。不老不死の伝承は、このコールドスリープ技術や細胞修復ナノマシンに由来する。 |

| **穢れ (Kegare)** | 放射性物質の汚染 / ナノマシンの暴走（バグ） | 死や病をもたらす不浄。本作では、旧世代のオーパーツや破損した動力炉から漏れ出す致死性の放射線、あるいはプロトコルから逸脱して生体組織を無秩序に分解・再構成しようとする暴走ナノマシン（グレイ・グー現象の初期段階）として扱う。 |
| **禊 (Misogi)** | 除染プロトコル / ナノ・スクラビング | 水で穢れを洗い流す行為。実際には、特定のナノマシン不活性化溶剤（神水）の塗布や、除染チャンバーにおける高エネルギー照射による汚染物質の除去、およびソフトウェア的なデバッグ・プロセスを指す。 |

### **2.2 File 03 (Characters) への追記案：史実の対立構造に基づくキャラクターダイナミクス**

史実における蘇我氏と物部氏の権力闘争1を、未知のテクノロジーに対する思想的アプローチの違いとしてキャラクター設定に深く落とし込む。この対立は、単なる善悪ではなく、「人類はいかにして高次存在（神＝AI）と向き合うべきか」という哲学的な問いをプレイヤーに投げかける。

**蘇我派閥（革新派・技術ハッカー集団）の深掘り:** 6世紀の史実において、蘇我稲目や蘇我馬子は渡来人と結びつき、新たな技術や思想である仏教を積極的に取り入れることで大和朝廷内で権力を拡大した1。この背景を拡張し、蘇我派閥のキャラクター群を「飛来したテクノロジー（仏教＝外部ネットワーク）の管理者権限を奪取しようとするハッカー集団」として設定する。彼らは、発掘されたオーパーツを解析し、自らの肉体にインプラント（サイボーグ化）することで超常的な能力を獲得している。彼らが携行する「念持仏」1は、戦術データリンクとパーソナルバリアを展開するAI端末であり、戦場における情報の優位性を確保する。彼らの究極の目的は、神々（旧AIネットワーク）の支配から人類を解放し、テクノロジーによる完全な自己進化（トランスヒューマニズム）を遂げることにある。彼らの態度は合理的で進歩的であるが、同時に倫理的タガが外れており、暴走のリスクを常に抱えている。

**物部派閥（保守派・古代兵器の継承者）の深掘り:** 対照的に、物部尾輿や物部守屋は日本古来の神祇と軍事を司り、外来の仏教を排斥しようとした1。これを基に、物部派閥を「古代から受け継がれてきた純正のオーパーツ（神祇兵装）を、ブラックボックスのまま使用する軍事エリート」として設定する。彼らの武器は、大気中のマナ（環境維持用ナノマシン）を収束させて放つ荷電粒子砲や、分子間結合を断つ振動剣など、圧倒的な物理破壊力を持つ。しかし、彼らはそれらの動作原理を理解しておらず、「穢れ（バグ）」を極度に恐れ、正しい儀式（パスワード入力と厳密なメンテナンス手順）を通じてのみオーパーツを使用する。彼らは蘇我氏の技術ハッキングを「世界の法則（テラフォーミング環境の維持）を乱す涜神行為」として断罪する。彼らの態度は高潔で伝統を重んじるが、変化に対応できず、システムの陳腐化とともに緩やかに滅びゆく運命にある。この対照的な二つの派閥が交錯することで、物語に重層的な政治ドラマが生まれる。

### **2.3 File 06 (Enemy) への追記案：妖怪・荒ぶる神の生物学的・システム的変異ロジック**

古代の文献に登場する「まつろわぬ民」や「異形の怪物」を、単なるファンタジーの産物としてではなく、「敗者の歴史」と「SF的変異」の交差点として生物学的・システム的に定義づける。

**土蜘蛛（Tsuchigumo）の変異ロジック:**

『古事記』や『風土記』において、大和朝廷に恭順しなかった先住民であり、手足が長く穴に住むと描写される土蜘蛛。彼らは元来、大規模な地殻変動や地下資源の採掘に従事していた自律型多脚重機（マイニング・ドローン）と、それに搭乗、あるいは中枢ユニットとしてサイボーグ化されて組み込まれた現住人類の末路である。テラフォーミングAIの指揮系統から外れ、自己防衛と「採掘・拡張」という初期指示だけを数百年にわたり繰り返し続けている。長い手足は油圧シリンダーや多関節の採掘アームの残骸であり、暗所での活動に特化した結果、本来の視覚器官は退化し、音響探知や振動センサーに置き換わっている。「糸を吐く」という伝承は、対象を拘束・捕獲するため、あるいは崩落を防ぐために射出される急速硬化性の化学ポリマーを指している。彼らは単なる悪役ではなく、システムに見捨てられ、狂った命令を実行し続ける労働者の成れの果てという悲哀を帯びた存在として描かれる。

**国栖（Kuzu）および鬼（Oni）の変異ロジック:**

山奥に住む異形の者、あるいは超人的な力を持つ先住民とされる国栖や鬼。彼らは、テラフォーミングの過程で生じた局地的な「高重力・高毒性環境」を切り抜け、特定のエリアを開拓するために、遺伝子工学によって意図的に生み出された環境適応型人類の末裔である。異常な筋力と、皮膚の硬質化（角質化による天然の装甲化）が施されている。彼らの頭部に生える「角」は、悪魔的な象徴ではなく、大気中の毒素（穢れ）をフィルタリングし、体内の余剰な熱や汚染物質を排出するための高度な放熱・排気器官である。大和朝廷を構成するベースライン人類からは、その異形の外見と圧倒的な身体能力ゆえに「鬼」として恐れられ、迫害と討伐の対象とされた。彼らの怒りは、造物主たるシステムと、恵まれた環境を独占するベースライン人類に向けられた正当な生存権の主張である。

**特異点としてのラスボス：ヒルコ（蛭子）:**

イザナギ・イザナミAIが最初に設計した「完全環境適応型・生命体プロトタイプ」。あらゆる極限環境（宇宙空間、深海、高放射能地帯）に適応するため、細胞の分化を固定せず、常に幹細胞の状態で環境に合わせて肉体を変化させるという野心的な設計であった。しかし、その「変化し続ける」性質ゆえに強固な骨格を形成できず（無脊椎動物のような形状に留まり）、高度な知性の統合にも失敗したと見なされた。結果として「失敗作」の烙印を押され、葦の舟（廃棄用ポッド）に入れられて軌道上、あるいは海溝の奥深くへと投棄された。しかし、極限環境の中でヒルコは生き延び、気の遠くなるような時間をかけて自己増殖と独自の進化を繰り返した。彼の目的は単なる破壊ではなく、「自身が生きられる環境への、地球の再テラフォーミング（現人類の絶滅と生態系の完全な再構築）」である。彼の目には、現在の地球環境こそが「誤った設定」に映っており、その行動論理には絶対的な正当性が存在する。

## **3\. システム改善プロポーザル (System Improvement Proposal)**

File 04 (Battle) におけるタイムライン・バトルシステムの課題を解決し、現代のプレイヤーの厳しい要求水準を満たすためのUI/UXおよび仕様の抜本的な改善案を提示する。

近年のRPG市場において、タイムライン制のバトルは高度な戦略性を提供する一方で、「次に誰が行動し、どのような連鎖的影響が出るのか」という情報の渋滞（認知負荷）を引き起こしやすいという構造的な課題を抱えている。また、競合タイトルである『崩壊：スターレイル』の分析2からも明らかなように、戦闘アニメーションの冗長性やテンポの悪さは、育成のための周回プレイにおいて致命的なストレス（プレイヤーが「しんどい」と感じる要素）となり、ゲームからの離脱を直接的に招く2。

### **3.1 タイムライン・バトルの視認性向上と情報の整理**

タイムラインバトルにおける情報の渋滞を防ぎ、直感的かつ深い思考を可能にするため、以下のUI設計とシステムアプローチを導入する。

| UI/UXにおける課題 | 既存アプローチの限界 | 本作での解決・改善案 (Actionable Proposal) |
| :---- | :---- | :---- |
| **行動順の視認性と予測可能性** | キャラクターの顔アイコンが1列に並ぶだけのUIでは、キャラクターごとの速度（Speed）の違いや、遅延効果（ディレイ）の相対的な影響が直感的に把握しにくい。 | **「動的相対タイムライン」の採用:** アイコンの単なる順序ではなく、行動発動までの「距離（時間的コスト）」を可視化する目盛り付きのレーンシステムを採用する。プレイヤーが特定の技（例：敵の行動遅延技）を選択した際、確定前に敵のアイコンがレーン上をどれだけ後退するかをゴースト（半透明のプレビュー）でリアルタイムに表示し、戦略的結果を視覚的に予測可能にする。 |
| **専門用語による認知負荷の増大** | 固有の神道・SF用語（例：禊、産霊、穢れ深度など）を用いたバフ/デバフ効果が分かりにくく、画面上に情報が溢れかえる。 | **「コンテクスト・ツールチップ」の実装:** 戦闘中、対象キャラクターや敵にカーソルを合わせる（スマホ版なら長押し、コンシューマー版なら特定ボタン押下）ことで、現在付与されているバフ/デバフのアイコンが拡大表示される。そこにフォーカスするだけで「専門用語のルビ」と「具体的な数値的影響（例：攻撃力-15%、継続ダメージ）」が瞬時にポップアップする。平常時の画面情報は最小限に留め、クリーンなUIを維持する。 |
| **連携技の爽快感と演出の分断** | 強力な連携技のたびに長いカットイン演出が挿入され、戦闘のテンポが著しく阻害され、周回時のストレスとなる。 | **「シームレス・オーバードライブ演出」:** 連携技発動時、画面が暗転して別の専用ムービー空間に切り替わる旧来の手法を廃止する。代わりに、通常戦闘のフィールド上でカメラがシームレスにズーム・回り込みを行い、リアルタイムでキャラクターが連続攻撃を叩き込む。背景の時間の流れはスローモーションになりつつ、攻撃自体は高速で処理されるため、爽快感とプレイのテンポを両立させる。 |

### **3.2 テンポ感の極限化とプレイヤー疲労の軽減**

競合のレビュー2にて「アニメーションが遅く、オートでもしんどい」「素材収集のスキップがなく、面倒が増えただけ」という強い不満が散見された。現代のプレイヤーは、可処分時間を奪う「意味のない待機時間」を極度に嫌う。これを払拭するため、以下の機能をエンジンレベルで実装する。

第一に、無劣化の「2倍・3倍速モード」の実装である。単純にゲームの再生速度を上げる（フレームを間引く）だけでは、モーションが不自然にカクつき、演出の魅力が損なわれる。これを解決するため、倍速時専用のアニメーション・ブレンディング（補間技術）を実装する。倍速時であってもモーションの繋がりが滑らかであり、かつ超高速で戦闘が進行する仕様を構築する。

第二に、「オーバーキル・スキップ」システムの導入である。プレイヤーのパーティと敵とのレベル差（あるいは戦力値の差）が一定以上ある場合、フィールド上で敵と接触した瞬間に「神機の一閃」のようなエフェクトが走り、戦闘画面に一切移行することなく、リザルト（経験値および報酬の獲得）のみがポップアップで表示されるシステムを導入する。これにより、旧エリアでの素材収集や低レベルの敵とのエンカウントにかかるストレスを完全にゼロにし、プレイヤーが「ストーリーの進行」や「高難易度ボスとの戦略的な戦闘」といった本質的な楽しみに集中できる環境を提供する。

## **4\. 矛盾・リスク指摘 (Risk Assessment)**

現在の企画および本レポートによる拡張案をゲームとして実装する上で、ナラティブの破綻やゲームメカニクスの崩壊を引き起こしかねない「致命的矛盾」や「懸念点」を、忖度なしで指摘し、その回避策を提示する。

### **4.1 歴史考証的矛盾のハンドリングとメタ構造化**

**懸念点:** 本プロジェクトは「飛鳥・奈良以前（主に5世紀〜7世紀）」を舞台としているが、この時期は日本における技術的過渡期である。史実において仏教伝来は6世紀半ば（欽明天皇期）とされている1。もしゲーム内で「神代（超古代のSFオーパーツ全盛期）」と「仏教伝来（外部ネットワークからの侵略）」を同時に描く場合、史実の年代設定（『古事記』が編纂された8世紀初頭までのタイムライン）との間に、数百年単位の時系列的なズレや圧縮が生じる。史実を重視するプレイヤー層からの反発や、世界観の説得力低下を招く恐れがある。

**対策方針:**

この歴史的な矛盾を、隠蔽するのではなく「ゲーム内の事実（異常事態）」として逆手に取る。作中において「何者かによって歴史が改竄、あるいは意図的に加速させられている」というメタ設定を組み込む。高次AI（神）や外部からの干渉により、本来数百年の時間をかけて普及するはずの技術（製鉄、土木、そして宗教という名のOS）が、わずか数十年で異常な速度で発展しているという不自然さを、作中のキャラクター（学者肌の渡来人や、過去の記録を持つアンドロイドなど）に明確に指摘させる。これにより、歴史考証の不整合への批判を回避しつつ、プレイヤーに「この世界の歴史はどこかおかしい」というSF的ミステリーのフックを提供し、物語の核心へと誘導する力に変換する。

### **4.2 長期運営・拡張におけるゲームバランスとインフレリスクの抑制**

**懸念点:** 『崩壊：スターレイル』などの長期的な運営や大型DLCの追加を見据えたRPGにおいて頻繁に指摘される問題として、キャラクターや装備品が追加されるにつれて「数値の計算が破綻し、誰にも適合しない死に装備」や「何をしても強くなれない初期キャラクター」が発生するリスクがある2。特に本作は「旧世代のオーパーツ（物理破壊）」と「新世代のネットワーク兵器（ハッキング・電子戦）」という、世代の異なる強力な兵器群が登場するため、設定上強力な新世代兵器を実装した瞬間に、初期のオーパーツ武器がすべてゲーム的価値を失う（完全な下位互換となる）危険性が極めて高い。

**対策方針:** 「垂直方向の強さ（純粋な攻撃力やHPといった数値のインフレ）」に依存したゲームバランスを廃し、「水平方向の強さ（環境適応性・特定のギミック解除能力）」によるバランス調整を徹底する。 例えば、最新の「蘇我氏系ネットワーク兵器」は基礎攻撃力は高いが、特定の「電磁パルス（穢れ）攻撃」を行う土蜘蛛系の敵に対しては、システムダウンを起こし完全に無力化するという弱点を設ける。その際、電子部品を持たず物理的な構造しか持たない「物部氏系オーパーツ（旧式武器）」が、逆に最適解（特効）となる、といったジャンケンに似た相互補完構造を構築する。 古い装備が「最新の環境や特定の運用において、逆に最適解になる」という現象2を、バグや計算ミスとしてではなく、ゲームデザインの仕様として意図的かつ緻密に組み込む。これにより、全装備・全キャラクターに固有の「役割（ニッチ）」が永続的に保証され、長期的なプレイにおけるプレイヤーの資産（育成したキャラクターや収集した装備）の価値毀損を防ぎ、徒労感を劇的に軽減できる。
