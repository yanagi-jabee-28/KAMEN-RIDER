**【プロジェクト・コア・フィロソフィー】**
* **タイトル:** アシブネノミコト —泥と星の神話—
* **テーマ:** 「完璧な神の理」に対する、「不完全な人間の熱量」の肯定。
* **デザイン原則:**
  * 神（敵・システム）は、傷を恐れ、無機質で完璧な予測（エントロピーの凍結）を行う。
  * 人（味方・プレイヤー）は、傷を歴史として引き受け、摩擦やエラー（ノイズ）で神の理を砕く。
* **すべての仕様は、「効率的な消費」ではなく「泥臭い修復と継承」をプレイヤーに体感させるために存在する。**

---

# RPG企画3 ドキュメント構成

本プロジェクトの仕様書は、役割分担を明確化した以下の4ファイルで構成する。

1. **01_Narrative_Design.md**
	- 世界観、物語、キャラクターの魂を規定する。
2. **02_System_Mechanics.md**
	- 戦闘、成長、経済のルールと骨格を規定する。
3. **03_Level_and_UX.md**
	- 空間設計、UI、プレイヤー体験の皮膚感覚を規定する。
4. **04_Data_Architecture.md**
	- 数値、計算式、データリストの血液を規定する。

---

## 運用ルール

- すべてのファイル冒頭に同一の「プロジェクト・コア・フィロソフィー」を置く。
- 仕様の重複記述を避け、各ファイルの責務境界を維持する。
- 物語（01）・ルール（02）・体験設計（03）・数値（04）を分離し、更新漏れを防ぐ。
