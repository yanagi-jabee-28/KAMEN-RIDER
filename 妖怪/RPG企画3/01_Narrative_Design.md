**【プロジェクト・コア・フィロソフィー】**
* **タイトル:** アシブネノミコト —泥と星の神話—
* **テーマ:** 「完璧な神の理」に対する、「不完全な人間の熱量」の肯定。
* **デザイン原則:**
  * 神（敵・システム）は、傷を恐れ、無機質で完璧な予測（エントロピーの凍結）を行う。
  * 人（味方・プレイヤー）は、傷を歴史として引き受け、摩擦やエラー（ノイズ）で神の理を砕く。
* **すべての仕様は、「効率的な消費」ではなく「泥臭い修復と継承」をプレイヤーに体感させるために存在する。**

# 01_Narrative_Design

## 1. 世界設定と神話解釈

- **神々の動機（エントロピーへの恐怖）:**
  神話的悲劇（イザナミの死、スサノオの暴走）を契機に、神々は「感情（ノイズ）こそが世界を滅ぼす」と結論づけた。結果として世界を無菌室のように凍結・管理しようとする。
- **対立構造:**
  - **天（天津神）:** 完璧、無機質、鏡面、対称。禊（スクラップ・アンド・ビルド）でエラーを消去する。
  - **地（国津神/人間）:** 不完全、泥臭さ、錆、非対称。金継ぎ（リノベーション）で傷を歴史として肯定する。
- **三貴子の配置:**
  - **アマテラス（敵・監視）:** 影を許さない絶対的な光。
  - **スサノオ（力・情念）:** 仲間が受け継ぐ「暴れる力」の源流。
  - **ツクヨミ（器・記録）:** ミコトが持つ「静かに溜め込む器」の性質。

## 2. キャラクターマトリクス

- **ミコト（主人公）:** 【器 / 形代】記憶喪失の「二隻目のアシブネ」。仲間のトラウマや穢れを肩代わりする。
- **うかみ:** 【指導者 / 猿田彦→役小角】斥候から行者へ変化。中盤離脱、終盤帰還。
- **ウズ:** 【道化 / 天鈿女命】予測不能な舞で神の計算を狂わせる。
- **マヒト:** 【職人 / 天目一箇神】隻眼の鍛冶師。傷つき使い込む鉄を愛する。
- **タチバナ:** 【反逆 / 弟橘媛】自傷ヒーラー。自己犠牲の美談を拒否する。
- **ワカヒコ:** 【観察者 / 天若日子】ハイリスクな狙撃手。
- **スクナ:** 【賢者 / 少名毘古那】毒と物質の専門家。

## 3. シナリオフローチャート（全4幕）

1. **第1幕（白の章）:** 天のゆりかごからの追放。うかみとの出会いと最初の痛みの転写。
2. **第2幕（泥の章/赤の章）:** ヤマト巡礼と葛城山の悲劇。うかみ離脱、ミコトの器損壊。
3. **第3幕（金の章・前半）:** 神脈断ち。残された仲間の覚醒とミコトの自我獲得。
4. **第4幕（金の章・後半）:** 黄泉比良坂で再会、行者帰還。別天津神・アマテラスと決戦、ヒルコ浄化。
