# 03_Level_and_UX

## 1. ワールド構造（採用版）

- **基本構造:** 周縁集落を巡礼して中心領域へ迫る円環導線。
- **主要エリア:**
  - **アマ（漁村）**: 漂着神信仰とつぎはぎ文化の導入。ここでミコトはうかみと出会い、赤いスカーフの儀式を経験する。
  - **マタギノ（狩猟村）**: 傷と贈与の文化、戦闘的生活圏。ここでマヒトと出会い、彼との静かな共鳴を経験する。
  - **ヌナカワ（泥の郷）**: トラウマを土偶化して祀る土地。ここではスクナの薬学的論争に巻き込まれ、彼の観察対象としての合流が始まる。
  - **タケミナカタの地**: 天への抵抗史を残す隠れ里。
  - **海辺の集落（生贄伝承）**: 澱んだ浜で怨霊扱いされたタチバナが立つ場所。ミコトたちはここで彼女と遭遇する。
  - **高所の砦**: ワカヒコが遠景から観察・試練を与える地点。後に決定的な介入を果たす。 
  - **葛城山**: うかみ離脱イベントの確定舞台。
  - **脈継ぎの泥舟:** 建てた社が連結されることで生まれる“ひび割れ”ネットワーク。マップ全体は陶片のように描かれ、ワープラインは漆で縫われた脈となる。移動はこの線上を泥舟で滑るように描写され、移動そのものがプレイヤーのアクションとして世界観を体現する。
## 2. 主要ダンジョン

- **ヤマト地下迷宮:** ヒルコ真相への到達地点。撤退導線と再突入導線を分ける。
- **無菌領域（ツクヨミ支配下）:** 中盤〜終盤の主要敵対エリア。ツクヨミの放つ「影（立体感）を消し去るフラットな照明」に取り憑かれた空間であり、プレイヤーは泥を塗って影を作ることで視覚・システム的な「死角」を作り対抗する。
- **黄泉比良坂:** 黄泉戸喫ギミックで通常回復が機能しにくい高圧リソース管理。
- **常世境界:** カガセオ戦を配置する試練領域。黄泉比良坂から合流したばかりの行者うかみを含めた5人パーティでの総力戦となる。**敵であるカガセオは、これまで彼らが武器の修復（金継ぎ）に使ってきた「星の砂」の本体（原初の反逆者）であり、自らの残骸・熱量を継ぐに足るかを見極める最大の壁として立ち塞がる。**
- **常世最深部:** 別天津神との最終決戦領域。この深部へ到達する前に天岩戸でアマテラスの凍結を解除する必要があり、プレイヤーは「恐れを抱いた管理者」の殻を破る儀式を経て、続いて「理そのものの根源」を粉砕するという二段階の体験を味わう。ここで正式にシステムルールを踏み越えた「5人共闘（うかみ自律）」が完全解禁される。
- **常世の運用方針:** 常世はボス戦専用の単線ステージではなく、探索・戦闘・資源管理を伴う第4幕の主要冒険領域として扱う。
  - **天岩戸（システム凍結エリア）:** 最深部手前に存在する「絶対障壁（管理者権限ロック）」。アマテラスが世界をフリーズさせた結果であり、ここではメニュー画面やHUDが「白一色（またはノイズ）」になり、通常の操作を受け付けない演出が入る。神器等を用いたイベント（狂騒の宴によるバグ・エラー注入）を起こすことで、UIにヒビが入り、システムが「再起動（泥臭いUIへの書き換え）」される。
- **傷跡の海（クリア後エンドコンテンツ）:** 「永遠（イワナガの理）」が崩壊し、正しく「死と劣化（サクヤの理）」が実装された世界でのパーシステント海域探索。もはや神でも帝でもない「ただの人間」となったミコトと仲間たちが、アシブネを拠点とし、未知の海域（グリッドやノード）を開拓する。サルベージポイント（無限の素材収集）や、過去の強敵の残滓が待ち受ける特異点（ボスラッシュ領域）が点在する。ここではうかみも「正規の4名枠（命令可能）」へ戻り、破格のシステム拡張から落ち着いた日常サイクル（金継ぎしながら限りある命を生きる営み）への意図的なクールダウンと、終わらない冒険を促す。

## 3. UX体験曲線

- **赤いスカーフの儀式:** 漁村アマで荒御魂を鎮めた後、村人から贈られた赤い布がうかみによってミコトの首に巻かれる。これはプレイヤーに「ここから英雄の旅が始まる」というビジュアルと感情のスタートラインを叩き込むイベントで、以降の道中でも度々語られる。
- **序盤:** うかみ依存の安定プレイを意図的に成立させる。
  プレイヤーは操作キャラ4人のうち、移動・先制・殴り合いを任せられる安心感を得る。UI上も彼のアイコンが常に最前列に固定されることで、自然に「頼る」感覚が育つ。
- **離脱直後:** うかみの突如の装備解除／消滅演出とともに画面の操作可能キャラが3人へと減り、メンバー切り替えUIが強調される。ミコトの神写しスキルが「うかみ技」に自動更新されるチュートリアルが挿入され、プレイヤーに「頼り続けてはいけない」心理的断絶を作る。また、控えとの再編を強いられる「扱いづらい戦線」において、ミコトが放つ『うかみの技』の存在感を極大化させ、「継承」のエモさをプレイヤーに体感させる。
- **帰還後（第4幕）:** うかみの再投入は控えスロットからではなく、メインUI上に不規則なノイズと共に「謎の5人目」アイコンとして飛び込む。その瞬間、タイムライン表示が5列に広がり、プレイヤーが制御を失うスペクタクルと安堵を同時に味わえる。
  - **決戦カタルシス:** 操作不能の5人目が勝手に戦果を挙げ、システム（敵の予告UI）を次々と破壊していく「怒涛のスペクタクル」と「心強さ」によるカタルシスの解放。
  - **導線制約:** 行者うかみ再加入直後から別天津神撃破まで、うかみのみ「常世・黄泉の国・黄泉比良坂」から離脱不可。パーティ全体は移動可能だが、領域外へ出る際はうかみを現地待機として扱う。
  - **ビジュアルバリエーション:** 特定カットシーンや外伝では、40歳前後の成熟したウカミ姿（渋イケメン）を採用し、プレイヤーに師匠としての威厳と安定感を見せることもできる。

## 4. UI仕様（確定）

※ UIの美術様式（縄文的アシンメトリー vs 弥生的シンメトリー）の詳細は `05_Art_and_Visual_Design.md` を参照。

- **タイムラインUI:** 神の次ターンの「最適化されたターゲット予告」を明示する。プレイヤーの自傷行動やうかみの乱入で予告UIに致命的なヒビが入る。**破壊時にはBGMの逆再生、文字化け、神のUIからの「謝罪」や「嘲笑」など、システムそのものがバグる演出を入れる。**
- **地図／ワープUI:**
  - 世界地図はひび割れた陶器を模したデザインで表示し、それぞれの社がひびの交点として刻まれる。
  - **社を建立した場所だけがワープ先として選択可能になる。既存の祠や社に御霊を奉納すれば即座にポイント化できる。**冒険初期は物資不足を補うため、祠に向かって選択肢を選ぶ「祈りUI」を表示し、成功すれば短時間のみワープ可能な仮拠点が設置される。社が無い地点にはファストトラベル機能は発生しない。
  - **風化の視覚化:** 頻繁に使用した仮設ラインは、黄金の輝きが褪せて「土気色（非アクティブ）」へと変化する。これを選択すると「再点火（コスト消費）」の確認ダイアログが表示され、承諾すると一瞬だけ強く輝いて移動が可能になる。
  - プレイヤーが社を建立するごとに、ひびに沿って恒久的な漆の線が走り、ワープ可能ポイントが増える。
  - ワープ選択時は現在地から目的地までのひび割れラインが黄金に輝きながら脈打つアニメーションを再生。**この黄金の輝きは、砕かれた星（カガセオ）の残骸が大地を繋いでいることの視覚的証明である。** 同時にシステム音が逆再生され泥の足音に置き換わる演出を実行。
  - VFXとSFXに合わせて、移動中の短いテキスト（ウズやタチバナなどの掛け合い）が表示され、キャンプスキット的な雑談を挟む。
- **神写しUI:**
  - 特技ごとの理解度ゲージを表示。
  - 習得済み特技と未習得特技を明確分離。
- **呼び継ぎ（キメラ）UI:**
  - 呼び継ぎを行った武器のステータス画面では、**ベース武器の名前に、素材となった「遺骨（元の武器名）」がノイズのように明滅・混入する演出**を入れる（例: `天叢雲剣` → `天叢...（鉄パイプ）...雲剣`）。
  - グラフィック上も、刀身の一部が明らかに異質な素材で継がれている「痛々しい違和感」を強調する。
- **剣塚の共鳴UI:**
  - 発動時、画面背後に「これまでのプレイで代受苦を行い、消失したすべての武器名」が高速でログとして流れ落ちる演出を採用。
- **パーティUI（負荷抑制）:** 最終決戦の5人目は「コマンド入力不要の自律キャラクター」として描画し、プレイヤーの操作対象UIは基本4名に据え置く。これにより過剰な認知負荷を防ぎつつスペクタクルを最大化する。
- **離脱制約UI（うかみ専用）:** 地図遷移時、対象領域外へ移動しようとすると「うかみは行者の誓約によりこの地を離れられない」という文言を表示し、うかみのみ待機処理へ自動遷移する。
- **金継ぎUI:** 破損履歴・付与候補・消費素材を同一画面で確認可能にする。**「代受苦」で折った武器の名前と履歴は拠点の「武具の墓標」リストに刻まれ、NPCがその喪失に言及する。**
- **インベントリ摩擦UI:** 負荷ランクと移動/先制への影響を常時確認できるようにする。
- **海域探索マップ（クリア後専用）:** 神の理（敷かれたレール）がない世界を、サルベージやボスラッシュを目指して自らの手で書き進め、開拓していく航海UI。

## 6. 神社建立イベント（進行/拠点拡張）

このセクションではプレイヤーがどのように神社建立を体験するか、操作フローとUI要素に焦点を当てる。

- **トリガー:** 各ボス（天津神・別天津神・裁定者）撃破後、固定の「御霊」がドロップし、インベントリに格納される。
- **拠点での反映:** 拠点メニューから「祀る」を選択すると建築フェーズへ遷移する。
- **建築フェーズ:** 
  * 初期敷地マップは**概念上10×10程度のブロック**として表現される。
  * **回路設計UX:** プレイヤーは「鳥居（IN）」を配置し、そこから「参道（LINE）」をドラッグして引き、「本殿（OUT）」へと接続する。
  * **シグナルフローの可視化:** 配置中、鳥居から光の粒子（信号）が流れ出し、参道を通って本殿へ届く様子がリアルタイムで描画される。
  * **インピーダンス警告:** 参道が途切れたり、フィルタ（手水舎）無しで直結すると、光が逆流して赤黒いノイズ（澱み）が発生するエフェクトを表示し、直感的に「悪い配置」であることを伝える。
  * **成功演出:** 美しい迂回ルートで接続が完了すると、回路全体が黄金に発光し、受容力ゲージが大きく上昇する。
- **UX体験:** 建築中は伝統音楽が流れ、2D俯瞰の設計UIがプレイヤーの思考を支援する。時間制限はなく、試行錯誤を楽しむ余地が残される。
- **進行との結びつき:** 受容力が閾値に達すると拠点の祓い力が上昇し、HP回復率・装備耐久回復率などの領域ボーナスが自動付与。神社拡張は次エリア解放の条件となる。
- **拡張と再設計:** いつでも社の改築が可能。新御霊を祀るたびに敷地を広げ、過去の配置を見直せる。
- **ビジュアル＆UI:** 建築フェーズは2D俯瞰表示。パーツパレットとバー表示で操作が直感的になる。
