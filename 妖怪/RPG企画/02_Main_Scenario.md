# 02 Main Scenario | メインシナリオ

## 序章：雷鳴と鏡の神

-   **漂着:** 記憶喪失の男「ミコト」が葦原の海岸に漂着する。
-   **異物の発見:** 海岸近くの洞窟で、ミコトのものと酷似しながらも数十年前に漂着したと見られる「壊れた葦船（カプセル）」を発見する。内壁には内側から掻き毟った痕と、骨のない生体の黒い残滓が残る。
-   **遭遇:** 時の帝（スメラノミコト）の一行が、未来兵器「雷筒（火縄銃）」で武装した謎の集団「土蜘蛛」に襲撃され、窮地に陥る。
-   **覚醒:** ミコトは、神としての本能で帝を守るために介入。護衛兵の「盾の構え」と、敵の「雷筒の扱い方」を瞬時に**『神写し』**し、敵を撃退する。
-   **勅命:** その異能を見出した帝は、ミコトを「国造り」の旅へと送り出す。それは、各地でオーパーツを用いて反乱を起こす勢力を鎮圧するための旅だった。監視役として、帝の忠臣である**武人（もののふ）**が最初の仲間となる。
-   **任意イベント：葦船洞の壁書き:** 序章終盤、再訪した葦船洞の壁面で、語り部が読めない文字列を採譜する。陰陽博士は後に「文字ではなく、反復する苦痛の記録に近い」と解釈する。
-   **テキスト伏線:** ミコトが葦船洞で待機すると、短い内語「ここは寒い」「まだ終わっていない」が低確率で発生する。

## 第二章：山越えの導きと策謀

-   **合流:** 魔の山「葛城山」を越えるため、帝が派遣した指揮官**兵部少輔（軍師）**と、現地案内人の**斥候（うかみ）**が仲間に加わる。
-   **対立と共闘:** 効率を重視する軍師と、ヤマトを憎む斥候、古い慣習を重んじる武人は反発しあう。しかし、山のヌシ（巨大な猪神）との戦いで、ミコトが彼らの技を繋ぎ合わせるように戦ったことで、一行は奇跡的に勝利。互いの力を認め合い、不本意ながらもチームとしての一歩を踏み出す。

## 第三章：穢れと禁忌の炎

-   **汚染:** 「常世の遺物」がもたらす「穢れ」によって腐敗した森へ。そこでは、人々が疫病に苦しんでいた。
-   **共鳴発作:** 汚染源や常世由来の兵器に近づくたび、ミコトは激しい頭痛と耳鳴りに襲われる。「痛い、寒い、寂しい」という正体不明の声が断続的に混線し、語り部はそれを「他者の怨嗟が器に響いている」と記録する。
-   **出会い:** 森を浄化しようと衰弱する**巫（かんなぎ）**と、汚染源ごと全てを焼き払おうとする狂気の**外法師（げほうし）**に出会う。
-   **解決:** ミコトは、外法師の「獄炎」をコピーし、巫の「聖なる結界」で守った場所から汚染源だけを精密に焼き払うという離れ業をやってのける。相反する力を持つ二人は、ミコトという「器」を通じて共存できることを知り、仲間に加わる。

## 第四章：歴史の編纂者と師匠

-   **遺跡:** 敵の技術の出処である「常世の門（古代遺跡）」へ。そこには朝廷の調査団がいた。
-   **漏言:** ミコトの戦果を聞いた帝は酒席で「今度こそ当たりか。最初のアレは海へ流すしかなかった」と口を滑らせる。兵部少輔は沈黙し、陰陽博士は記録だけを残して真意を伏せる。
-   **再会と加入:** 調査団の長である**陰陽博士（はかせ）**は、外法師のかつての師匠だった。彼はミコトという「理論で説明できない存在」に学術的興味を持ち、同行を決意する。
-   **最後の仲間:** 遺跡の最深部は「言霊」の封印で閉ざされていた。そこに、これまで一行の噂を広めていた謎の吟遊詩人**語り部（かたりべ）**が現れ、祝詞で封印を解く。「この物語の結末を見届ける」と宣言し、8人目の仲間となる。
-   **任意イベント：禁書庫の断簡:** 朝廷の禁書庫で「淡島計画・第一試作」という断簡を入手できる。本文は大半が削除されているが、「歩行不能」「海上廃棄」「回収失敗」の語だけが読める。
-   **異常史イベント（追補）:** 陰陽博士が「本来数世代かかるはずの技術普及が数十年で進んでいる」と記録し、歴史加速の異常を示唆する。

## 第五章：国譲りと星の反逆

-   **対立の顕在化:** 土蜘蛛と荒ぶる神の二勢力が、同時期に同調したような異常攻勢を開始する。
-   **真因:** 中核には「まつろわぬ神」カガセオの干渉があり、彼は天秩序への服属そのものを拒んでいると判明する。
-   **決戦:** 一行は宿魂石祭壇でカガセオと対峙し、完全消滅ではなく封印再起動によって退ける。
-   **帰結:** カガセオはヒルコ配下ではない独立敵対者であることが確定し、敵対理由の違いが明示される。

## 終章：神代の終わり

-   **正体:** 敵の首魁は、ミコトと同じ「現人神」のプロトタイプでありながら、不具であったために廃棄された**『ヒルコ（失敗作の兄）』**だった。彼はヤマトへの復讐と、自らを捨てた母（生体プラント「イザナミ」）への思慕から、機械の要塞**『ヤマタノオロチ』**を起動させる。
-   **暗躍の回収:** 旅路で発生した兵器流通・反乱連鎖・情報撹乱の多くは、ヒルコが終章決戦へ誘導するために仕掛けた工作だったと判明する。
-   **決戦:** ミコトたちは仲間との連携でオロチを破壊。ミコトは深層部でヒルコと対峙し、「神写し」対「神喰らい」の激闘の末、兄の魂を鎮める。ミコトは役目を終え、歴史の表舞台から姿を消す。

## エピローグ：古事記の編纂

-   **数十年後。** 年老いた語り部が、朝廷でミコトたちの冒険譚を献上している。
-   **改竄:** しかし、時の権力者の意向により、その生々しい「事実」は、国家の威信を示すための「神話」へと書き換えられていく。
    -   巨大要塞オロチ → ヤマタノオロチ
    -   要塞攻略戦 → 天岩戸隠れ
    -   ミコトと仲間たちの活躍 → スサノオ、オオクニヌシ、その他多くの神々の逸話へ
-   プレイヤーだけが「神話の裏に隠された真実」を知る者となり、物語は幕を閉じる。
