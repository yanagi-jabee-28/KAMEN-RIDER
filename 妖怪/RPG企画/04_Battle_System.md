# 04 Battle System | 戦闘システム

## 1. 基本システム

-   **形式:** タイムラインバトル（行動順到達時にコマンド選択）
-   **パーティ:** 最大8人の中から、戦闘に参加する4人を選抜する「4人選抜制」。戦闘中にメンバー交代も可能（兵部少輔のスキルで効率化）。

### 1.1 行動フロー（タイムライン方式）

-   各ユニット（味方・敵）は `行動値` を持ち、時間経過で増加する。
-   `行動値` が閾値に到達したユニットから行動権を得る。
-   行動権を得た瞬間に、プレイヤーはそのキャラの行動を選択する（事前に全員分を予約しない）。
-   行動後は技ごとの `待機時間（WT）` を加算し、次の行動順が再計算される。
-   **敏捷** は `行動値の増加速度` と `WT軽減` の両方に影響する。

### 1.2 設計上の狙い

-   盤面変化（敵の詠唱、味方の被弾、神懸かり発動）を見てから最適行動を選べるため、判断の手触りが向上する。
-   「神写し」「連携技」「神懸かり」の発動タイミング管理が戦術の核になり、各コアシステムと噛み合う。
-   行動予約型よりテンポが落ちにくく、4人選抜でも待ち時間のストレスを抑えやすい。

### 1.3 タイムライン化に伴う整合ルール（必須）

-   **効果時間の基準統一:** 「nターン」は使用しない。継続効果は以下のどちらかで管理する。
    -   `行動回数基準`（例: 対象の次の2回行動まで）
    -   `時間基準`（例: 300CT 経過まで）
-   **バフ/デバフの標準:** 原則は `対象の行動回数基準` を採用する（高速キャラの一方的優位を抑えるため）。
-   **連携技の成立条件:** 発動者の行動時に、参加者が「行動可能」または「連携待機状態」である場合のみ成立する。
-   **連携技の成立条件:** 連携対象2名のうち、どちらのキャラクターの行動時でも発動できる。
-   **行動消費ルール:** 連携発動時に行動を消費するのは発動者のみとし、参加者の行動権は消費しない。
-   **参加者側の制約:** 参加者は戦闘不能・拘束・行動不能状態では連携参加できない（行動権の有無は問わない）。
-   **メンバー交代の処理:** 交代は行動1回を消費し、交代した側・入る側の両方にWTを付与する。
-   **同時到達の優先順:** 同じ時刻に複数ユニットが閾値到達した場合は、`敏捷 → 現在HP割合 → ランダム` で決定する。
-   **ゲージ増加の上限:** 神気ゲージ増加は短時間での連続加算に減衰をかけ、高速連打による暴走を防ぐ。

## 1.5 ステータス設計（採用版）

本作の基礎ステータスは、以下の12項目を正式採用する。

| 分類     | ステータス名       | 役割・効果（設計意図）                       |
| :------- | :----------------- | :------------------------------------------- |
| 基礎     | **体力（HP）**     | 戦闘不能判定に直結する生存値。0で戦闘不能。  |
| 基礎     | **魔力（MP）**     | 術法・特技・連携技の消費リソース。           |
| 物理     | **攻撃力**         | 物理系スキル・通常攻撃のダメージ基礎値。     |
| 物理     | **防御力**         | 物理ダメージの軽減値。高いほど被ダメージ減。 |
| 行動     | **敏捷**           | 行動順・再行動タイミングに影響する速度値。   |
| 会心     | **クリティカル率** | 物理・一部特技の会心発生確率。               |
| 命中回避 | **命中率**         | 攻撃・状態異常付与の成功判定に寄与。         |
| 命中回避 | **回避率**         | 単体攻撃・一部全体攻撃の回避判定に寄与。     |
| 魔法     | **魔法攻撃力**     | 属性術・呪術・陰陽術などの威力基礎値。       |
| 魔法     | **魔法防御力**     | 魔法系ダメージの軽減値。                     |
| 回復     | **回復力**         | 回復術・回復連携の回復量に影響。             |
| 耐性     | **異常耐性**       | 毒・麻痺・混乱・呪い等の付与確率を低下。     |

### ステータス運用ルール

-   バフ／デバフは原則として上記12項目にのみ付与する（効果の可読性を優先）。
-   「蘇生性能」「浄化性能」などの個別性能は、独立ステータスではなくスキル固有効果で表現する。
-   回復職の伸びしろは、**回復力**と**魔力（MP）運用**で差別化する。
-   物理職と術者職の差別化は、**攻撃力／魔法攻撃力**の配分と、**敏捷**で制御する。

### 耐性・弱点の統一仕様（倍率レベル方式）

耐性と弱点は別システムにせず、同一の「倍率レベル」で管理する。

-   管理対象: 属性（火/水/雷/風/土/光/闇）、状態異常（毒/麻痺/混乱/呪い/睡眠/沈黙）、物理種別（斬撃/打撃/刺突）
-   各対象に対して、`耐性レベル` を設定する（負=弱点、正=耐性）
-   例: `-2` は強い弱点、`0` は等倍、`+2` は強い耐性

#### ダメージ計算への反映

-   最終倍率は以下で算出する。  
    `最終倍率 = clamp( 1.0 - (耐性レベル × 0.25), 0.10, 2.00 )`
-   変換例:
    -   `耐性レベル = -2` → 1.50倍（弱点）
    -   `耐性レベル = -1` → 1.25倍（弱点）
    -   `耐性レベル = 0` → 1.00倍（等倍）
    -   `耐性レベル = +1` → 0.75倍（耐性）
    -   `耐性レベル = +2` → 0.50倍（耐性）
    -   `耐性レベル = +3` → 0.25倍（強耐性）
-   完全無効（0倍）は原則禁止。最低保証ダメージを残し、戦術の固定化を防ぐ。

#### 付与率（状態異常）への反映

-   状態異常は「付与成功率」に同じレベルを適用する。  
    `最終付与率 = 基礎付与率 × clamp( 1.0 - (耐性レベル × 0.20), 0.10, 2.00 )`
-   これにより、耐性が高い敵にも低確率で通る余地を残し、完全封殺を避ける。

#### レベルの合算ルール

-   `最終耐性レベル = キャラ素養 + 防具補正 + バフ/デバフ補正`
-   推奨上限: `-3 ～ +3`
-   防具は主に `+1 ～ +2` を供給し、弱点打ち消しは可能だが「完全無欠化」は難しい設計にする。

## 2. コアシステム①：『神写し（かみうつし）』

-   **使用者:** 主人公 **ミコト**
-   **概要:** ミコトは固有の戦闘スキルをほとんど持たない。代わりに、仲間や敵の技を「模倣」して戦う。
-   **スキルセット:**
    -   メニュー画面で、パーティにいる仲間のスキルを最大4つまで「装備」できる。装備したスキルは、戦闘中にミコト自身のコマンドとして使用可能。
    -   ただし、威力は本家の8割程度、MP消費は少し多いなど、完全な上位互換にはならない。
-   **ラーニング:**
    -   敵（特にボス）が使う特殊な「オーパーツ技（銃撃やレーザーなど）」を受けた際、その戦闘中限定でその技をコピーして撃ち返せる。

## 3. コアシステム②：『開眼（かいがん）』

-   **概要:** 仲間が上位スキルを習得する成長システム。
-   **条件:** ミコトが『神写し』で仲間の技を使い続ける（一定回数使用する）。
-   **効果:** 模倣された技のオリジナルである仲間が、「なるほど、あの技はそう使うのか」と戦闘中に閃き、より強力な**「進化・派生技」**や**「奥義」**を習得する。
-   **コンセプト:** 「主人公が真似をすることで、仲間が強くなる」という独自の育成サイクル。プレイヤーは仲間を強化するために、積極的にミコトで技をコピーすることになる。

## 4. コアシステム③：『神懸かり（かみがかり）』

-   **概要:** 戦闘中にキャラクターがトランス状態に入る特殊覚醒システム。いわゆる「ゾーン」や「バースト」に相当する。
-   **発動条件:**
    -   各キャラは非表示の**「神気ゲージ」**を持つ。
    -   ダメージを受ける、クリティカルを出す、敵を倒す、などの行動でゲージが上昇。
    -   特に、**キャラクターの役割（ロール）に合った行動**（武人なら「かばう」、巫なら「回復」など）で大きく上昇する。
    -   ゲージがMAXになると、自動的に**『神懸かり』**状態（体が光るオーラに包まれる）に移行する。
-   **効果:**
    -   発動者の**次の2回行動まで**、全ステータスが大幅にアップ。
    -   状態異常を無効化。
    -   **強力な「連携技」が使用可能になる。**

## 5. コアシステム④：『連携技（れんけいわざ）』

-   **概要:** 複数のキャラクターが協力して放つ合体技。戦術の要。
-   **発動主体:** 対象2名のうち、どちらの行動順でも起動可能。
-   **行動消費:** 連携を選択した発動者のみ行動を消費し、もう一方は行動を保持する。
-   **カテゴリ:**
    1.  **【阿吽の呼吸】（通常連携）:**
        -   **条件:** MPを消費するだけで、いつでも発動可能。
        -   **内容:** 物理的な合体攻撃や、戦術的な魔法の組み合わせなど。
    2.  **【神遊び（かみあそび）】（神懸かり連携）:**
        -   **条件:** 参加メンバーのうち**1人以上**が『神懸かり』状態であること。
        -   **内容:** 神の力を借りた奇跡や、超常現象。
        -   **コスト:** 発動後、起点となったキャラクターの『神懸かり』状態は解除される。
-   **連携の分岐（阿吽システム）:**
    -   同じキャラクターの組み合わせでも、「攻撃技」同士を重ねるか、「補助技」同士を重ねるかで、発動する連携技が変化する。

## 6. コアシステム⑤：『四魂の陣（しこんのじん）』

-   **概要:** 汎用の4人連携技。パーティ構成を問わず、いつでも最大火力を狙える。
-   **発動条件:** 戦闘に参加している**4人全員**が『神懸かり』状態であること。
-   **コスト:** 発動後、**全員の『神懸かり』が解除**され、神気ゲージがゼロになる。
-   **選択可能な効果:**
    -   **【荒魂】:** 敵単体に防御無視の超絶ダメージ。
    -   **【奇魂】:** 味方全体に「被ダメージ0」を**次の1回行動まで**付与。
    -   **【和魂】:** 味方全員のHP/MP全回復＋蘇生。
    -   **【幸魂】:** 味方全員に「ステータス大幅上昇＋消費MPゼロ」を**次の2回行動まで**付与。

## 7. 脅威システム：『穢れ（ケガレ）』

-   **概要:** 「常世の遺物（オーパーツ）」がもたらす汚染。
-   **効果:**
    -   オーパーツ系の敵と戦ったり、汚染されたフィールドを歩くと**「穢れゲージ」**が蓄積する。
    -   ゲージが溜まると、**最大HPが減少**したり、回復魔法の効果が半減するなどのデメリットが発生する。
    -   **浄化:** このゲージは、**巫（かんなぎ）**の特定のスキルか、町の社（やしろ）でしか下げることができない。これにより、巫のパーティ内での重要性が増す。
