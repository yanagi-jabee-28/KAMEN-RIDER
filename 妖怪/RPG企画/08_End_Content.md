# 08 End Content | クリア後要素

## 1. クリア後シナリオ：『黄泉の国・風土記』

-   **開始条件:** 本編エンディング後、語り部（かたりべ）に話しかける。
-   **あらすじ:** 「実は、帝には語らなかったもう一つの冒険がありましてな…」と切り出され、世界の歪みの根源を断つための最後の旅が始まる。
-   **新ダンジョン：【常世の座（とこよのくら）】**
    -   **場所:** ミコトが最初に漂着した海岸の地下深くに開いた次元の裂け目。
    -   **設定:** オーパーツの発生源である、未来の宇宙船「箱舟」の残骸。古代の岩盤と未来の金属・配線が融合したバイオメカニカルな迷宮。

## 2. 裏ボス

### ① 機神・天照 / 機神・月読

-   **場所:** 「常世の座」の中ボス。
-   **正体:** ミコトの「オリジナル」となった、旧世代の生体兵器のプロトタイプ。
-   **特徴:** 極大熱線を放つ「天照」と、パーティメンバーをコピーして戦わせる「月読」。

### ② 真の裏ボス：別天津神（モデル『天の御柱』）

-   **場所:** 「常世の座」の最深部（ブリッジ）。
-   **正体:** 箱舟のメインAIであり、歴史の管理者。
-   **動機:** ミコトたちの活躍により、文明レベルが規定値を超えたと判断し、「文明のリセット（大浄化）」を実行しようとする。
-   **特徴:** 感情を持たない幾何学的な光の集合体。「歴史改変（CT巻き戻し）」や「強制排除（メンバーロスト）」など、ルール介入型の攻撃を仕掛けてくる。

## 2.5 追憶イベント：『葦船洞・再訪』

-   **開始条件:** ヒルコ撃破後、語り部と陰陽博士を編成して葦船洞に入る。
-   **内容:** 序章で発見した壊れた葦船の内部に、当時は読めなかった刻印と爪痕の層が現れる。
-   **開示テキスト:** 「第一試作個体、海上廃棄。回収不能。生体反応、継続。」
-   **演出:** ミコトが短時間だけ共鳴し、ヒルコの幼い独白（痛みと寒さ）を追体験する。
-   **報酬:** 記録品「淡島断簡」と、ミコト専用の補助装備「流寓の首環（共鳴耐性＋）」を獲得。

## 3. エンドレスコンテンツ：『真・百鬼夜行（無限回廊）』

-   **開始条件:** 裏ボス撃破後。
-   **目的:** 裏ボス報酬の「神代の遺産（チート装備）」を使い、インフレしていく敵を蹂躙するサンドボックス・ダンジョン。
-   **ルール:**
    -   ランダム生成されるダンジョンを、死ぬまで潜り続けるサバイバルモード。
    -   10階層ごとにボスが出現し、休憩所（セーブ・回復）がある。
-   **特殊な環境:**
    -   **「灼熱地獄」:** 一定CTごとにダメージ。
    -   **「因果逆転」:** 回復がダメージになる。
    -   **「重力崩壊」:** 戦闘開始時、敵側の行動値が大きく加算される。
-   **特殊な敵:**
    -   **『無限再生スライム』:** 規定CT内に倒さないとHPが全回復する。
    -   **『鏡面・八咫烏』:** 物理ダメージを10倍にして反射する。
    -   **『写し身（ドッペルゲンガー）』:** 100階層ごとに現れる、現在のプレイヤーと全く同じ装備・ステータスのパーティ。
-   **報酬:**
    -   **『神珠』:** ステータス上限を突破する強化アイテム。
    -   **『常世の図鑑』:** 現代のガラクタ（スマホ、ミニカー等）を集めるコレクション要素。
    -   **『神の衣』:** 見た目が変わるスキン装備（セーラー服、スーツなど）。

## 4. 真エンディング（シークレット）

-   **条件:** ヒルコを倒し、無限回廊の最深部（例: 1000階）に到達する。
-   **内容:**
    -   数千年後の現代日本。とある神社の境内で、少年（ミコトの生まれ変わり）が、8人の名前が刻まれた古びた石碑を見つける。
    -   少年が空を見上げると、一瞬だけ、かつての仲間たちの笑い声が聞こえる。
    -   画面に**「神代は終わり、歴史は続く」**と表示されて終了。
