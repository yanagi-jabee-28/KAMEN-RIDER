# 05 Skill Database | 特技・術法全書

## 1. 稀人（ミコト）

| 分類     | 技名                   | 効果                                                                   |
| :------- | :--------------------- | :--------------------------------------------------------------------- |
| **固有** | 神写し（かみうつし）   | 装備している仲間の技を使用する。                                       |
| **固有** | 天津罪・国津罪         | 敵の特殊技を受けた後、次の自分の行動時に倍の威力で撃ち返すカウンター。 |
| **防御** | 八咫鏡（やたのかがみ） | 次の自分の行動開始まで、あらゆる攻撃を1回だけ無効化・反射する。        |
| **隠密** | 虚空（こくう）         | 次の自分の行動開始まで、敵の単体攻撃の対象にならなくなる。             |
| **補助** | 神の息吹               | 自分のHPを消費し、味方一人のMPを回復する。                             |
| **自動** | 合わせ鏡               | 敵が「自分と同じ技」を使った時、即座に無効化する。                     |
| **奥義** | 八百万・神集い         | 仲間の最強技を1人で連続で叩き込む。                                    |
| **探索** | 神籬（ひもろぎ）       | ダンジョン内に簡易セーブ・回復ポイントを作成する。                     |

## 9. 連携技（クロス・スキル）

特定の2人以上がパーティにいる時のみ使用可能。MP消費／条件（神懸かりなど）によって発動する、関係性を象徴する合体技群。

### A. 【阿吽（あうん）の呼吸】（通常連携）

| 分類 | 技名                          | 効果（概要）                                                              |
| :--- | :---------------------------- | :------------------------------------------------------------------------ |
| 物理 | 双璧・十文字（ミコト×武人）   | 敵単体に物理大ダメージ＋スタン。基本のクロスコンボ。                      |
| 物理 | 金剛・飛礫（武人×行者）       | 武人を投げて敵集団へ特大ダメージ＋ガードブレイク。                        |
| 防御 | 影法師の盾（武人×行者）       | 武人が盾となり物理無効化＋カウンターで反撃（スタン付与）。                |
| 戦術 | 伏兵・山颪（軍師×行者）       | 罠設置＋規定CT経過時に範囲大ダメージ・足止め。                            |
| 魔法 | 焦土作戦（博士×外法師）       | フィールドを可燃化→火で着火。全体炎ダメージ＋継続炎上（素早さ低下付与）。 |
| 支援 | 禊ぎ火（外法師×巫）           | 敵にはダメージ、味方にはHP回復効果を与える“浄化された炎”。                |
| 補助 | 英雄・偶像化（語り部×武人）   | 武人の攻撃・防御2倍（短時間）＋バーサクによる操作不能リスクを伴う。       |
| 妨害 | 虚報・四面楚歌（語り部×軍師） | 敵全体を混乱（同士討ち誘発）＋防御ダウン。                                |
| 知略 | 天地明察（軍師×博士）         | 弱点解析→味方全員の次の1回行動が弱点属性かつクリティカルになる。          |

### B. 【神遊び（かみあそび）】（神懸かり連携）

※発動条件：参加者のうち少なくとも1人が『神懸かり』であること（技により全員条件あり）。

| 分類     | 技名                         | 効果（概要）                                                                 |
| :------- | :--------------------------- | :--------------------------------------------------------------------------- |
| 召喚系   | 火生三昧（外法師×行者）      | 鬼神召喚に獄炎付与。次の3回行動まで攻撃追撃（召喚獣強化）。                  |
| 結界     | 清浄結界・五芒星（巫×博士）  | 魔法ダメージ無効＋MP吸収（変換）で魔法ボスへの最強カウンター。               |
| 即死系   | 鎮魂歌（巫×語り部）          | アンデッド・霊体に即死（昇天）。フィールドで味方HP継続回復。                 |
| 超火力   | 煉獄・雷公（外法師×博士）    | 獄炎＋雷の複合プラズマ攻撃。防御無視の極大ダメージ。                         |
| 単体殲滅 | 神去り・神生み（ミコト×巫）  | ミコトが神の一撃を放ち、敵単体を消滅させる（ボスにも超ダメージ／反動あり）。 |
| 終戦術   | 国譲り（軍師×語り部×ミコト） | 戦闘を強制的に撤退扱い（戦闘終了＝勝利扱い）。条件・制限付きの戦略技。       |

### C. 三人以上の伝説級連携（トリオ／トリニティ）

- 三種の神器（ミコト×武人×巫） — 防御無視の聖属性・超特大ダメージ＋味方全回復（テンション最大消費）。
- 百鬼夜行（外法師×語り部×行者） — 全体ダメージ＋ランダム状態異常（毒・混乱・恐怖など）。
- 森羅万象・混沌（博士×外法師×行者） — フィールド破壊的ギャンブル技（敵全滅 or 味方全HP1などの乱数効果）。

### D. 【四魂の陣】4人連携（グランド・コンボ）

| 技名（四魂）                 | 効果（概要）                                                                                      |
| :--------------------------- | :------------------------------------------------------------------------------------------------ |
| 荒魂・国崩し（あらみたま）   | 出ている4人の攻撃力＋魔法力を合算した防御無視の超特大ダメージ（フィニッシャー）。                 |
| 奇魂・天岩戸（くしみたま）   | 発動後、次の味方先頭行動まで敵のあらゆる攻撃を完全無効化（ダメージ0）。ボスの溜め技対策に最適。   |
| 和魂・大祓（にぎみたま）     | 味方全員のHP/MP全回復＋戦闘不能者の完全復活＋全デバフ解除。                                       |
| 幸魂・天地開闢（さきみたま） | 味方全員の次の2回行動まで、攻撃・防御・素早さを最大バフにし、技のMP消費をゼロにする（強化特化）。 |

> 備考：
> - 通常連携（阿吽）はMPで発動可。神遊び（神懸かり連携）は『神懸かり』状態が起点。4人連携は条件が厳しく演出重視の切り札。
> - UI上で「今使える連携」を表示し、同一ペアでも選択したコマンドによって発動する連携が変化する設計を推奨。
