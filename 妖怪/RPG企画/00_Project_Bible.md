# 00 Project Bible | 企画概要書

## 1. コンセプト

-   **タイトル案:** 『神代（かみよ）の終わり、古事記（ふることふみ）の始まり』
-   **ジャンル:** 歴史改変・神話創生RPG
-   **テーマ:** 「神話は、泥臭い人間たちの戦いから生まれた」という「事実」を追体験する物語。
-   **世界観:**
    -   飛鳥・奈良時代以前の、神と人、鉄と呪術が混在する古代日本（ヤマト）。
    -   「常世（とこよ）」から流れ着く未来の漂着物（オーパーツ）により、歴史が歪み始めている世界。

## 2. ストーリー概要

記憶を失った男「ミコト」は、その身に宿る異能「神写し（技の模倣）」を買われ、帝（スメラミコト）より「国造り」の勅命を受ける。
それは、未来兵器で武装する反乱豪族や、汚染で変異した荒ぶる神々を鎮める過酷な旅であった。
ミコトは7人の仲間たちと共に、自らの出生の秘密と、この世界の歪みの根源に迫っていく。
彼らの戦いの記録は、後に脚色・編纂され、日本最古の歴史書『古事記』として結実する。プレイヤーだけが、その「神話の真実」を知る目撃者となる。

## 3. 主要キャラクター

-   **ミコト（稀人）:** 主人公。仲間の技を模倣する「神写し」の能力を持つ、記憶喪失の現人神。
-   **武人（もののふ）:** 盾役。物部氏の生き残り。鉄壁の守りを誇る。
-   **行者（ぎょうじゃ）:** 速攻・召喚役。元は斥候だったが、修行の末に鬼神を使役する修験者へと覚醒する。
-   **外法師（げほうし）:** 破壊役。炎に特化した呪術師。自らの命を削って超火力を放つ。
-   **巫（かんなぎ）:** 回復役。唯一の完全回復・蘇生能力を持つ癒やし手。
-   **語り部（かたりべ）:** 支援・妨害役。言霊で戦場を支配し、味方のMPを管理する。
-   **陰陽博士（おんみょうはかせ）:** 弱体・属性役。五行の理で敵の弱点を突く賢者。外法師の師匠。
-   **兵部少輔（ひょうぶのしょう）:** 指揮役。通称「軍師」。味方を再行動させる司令塔。

## 4. コアシステム

-   **神写し（カミウツシ）:** 主人公ミコトが、仲間のスキルを装備して使ったり、敵の技をラーニングするシステム。
-   **開眼（カイガン）:** ミコトが技を真似ることで、オリジナルの仲間が刺激を受け、上位スキルを習得する成長システム。
-   **神懸かり（カミガカリ）:** 戦闘中にテンションが最大になると発動するトランス状態。ステータスが向上し、強力な「連携技」が使用可能になる。
-   **連携技（レンケイワザ）:** 特定のキャラクターの組み合わせで発動する合体技。「通常連携」と、神懸かり状態が必要な「神遊び連携」の2種類が存在する。
-   **四魂の陣（シコンノジン）:** 4人全員が神懸かり状態で発動できる超必殺技。「攻撃」「防御」「回復」「強化」の4つの効果から選べる。
-   **常世の遺物（オーパーツ）:** 敵が使う未来兵器（火縄銃、湾刀など）。プレイヤーも使えるが、「穢れ」のデメリットがある。

## 5. エンドコンテンツ

-   **裏ボス:** 世界の根源たるシステム「別天津神」と、ミコトの失敗作である「ヒルコ」との決着。
-   **無限回廊（百鬼夜行）:** クリア後に解放されるエンドレスダンジョン。裏ボス報酬のチート装備を使い、インフレしていく敵を蹂躙する。

## 6. ディープリサーチ反映方針（追補）

-   **神話×SF二重記述:** 用語は神話表現と技術実体の両方で説明し、没入と理解を両立する。
-   **思想対立の主軸化:** 蘇我系（革新）と物部系（保守）の対立を、宗教対立ではなく技術倫理対立として描く。
-   **敵の悲劇性:** 土蜘蛛・鬼・ヒルコは「排除すべき怪物」ではなく、システム不適合で生まれた敗者として描写する。
-   **快適性優先の戦闘設計:** タイムライン可視化、倍速、周回スキップを初期仕様で担保する。
