ドラゴンクエストシリーズに登場する呪文、特技、れんけい技、超絶技、必殺技、敵専用技などを、似た効果のものごとに分類し、網羅的に列挙した大辞典を作成しました。※同じ技でも作品によって効果や仕様が異なる場合は、その旨を記載しています。

### 1\. 攻撃呪文

**【火・炎属性】**

* **メラ / メラミ / メラゾーマ：** 敵単体に小さな火の玉〜巨大な火柱で炎属性ダメージ。  
* **メラガイアー：** メラ系の最上位。敵単体を巨大な火柱で焼き尽くす。  
* **メラストーム：** 敵を選ばず小さな火の玉を4連射する。  
* **ギラ / ベギラマ / ベギラゴン：** 敵1グループに炎や火炎でダメージ。  
* **ギラグレイド：** ギラ系の最上位。敵1グループを地を揺るがす炎で焼き払う。

**【爆発・雷属性】**

* **イオ / イオラ / イオナズン：** 敵全体に爆発を起こしてダメージ。  
* **イオグランデ：** イオ系の最上位。想像を絶する爆発で敵全体を吹き飛ばす。  
* **デイン / ライデイン / ギガデイン：** 敵1体、1グループ、または全体に稲妻を落とす。（※作品により攻撃範囲が単体・グループ・全体と異なる）  
* **ミナデイン：** 味方全員のMPを消費し、絶大な雷を落とす。

**【氷・水属性】**

* **ヒャド / ヒャダルコ：** 敵単体〜グループに氷の刃でダメージ。  
* **ヒャダイン：** 敵全体に氷のダメージ。（※シリーズ初期〜中期にかけて全体攻撃として登場）  
* **マヒャド / マヒャデドス：** 敵全体を巨大な氷の柱などで攻撃。  
* **コーラルレイン / メイルストロム：** サンゴの欠片や巨大な水流で全体やグループを攻撃。

**【風属性】**

* **バギ / バギマ / バギクロス：** 敵1グループ（作品によってはランダム複数）を竜巻で切り裂く。  
* **バギムーチョ：** バギ系の最上位。極大の竜巻で敵を切り刻む。

**【闇・土属性】**

* **ドルマ / ドルクマ / ドルモーア / ドルマドン：** 敵単体に闇の力でダメージ。  
* **ジバリア / ジバリカ / ジバリーナ / ジバルンバ：** 敵単体〜グループに土属性の魔法陣を仕掛け、行動時に爆発してダメージを与える。

**【即死・究極・その他】**

* **ザキ / ザラキ / ザラキーマ：** 呪いの言葉で敵を即死させる（単体 / グループ / 全体）。  
* **メガンテ：** 自分の命と引き換えに敵全体を即死または瀕死にさせる。  
* **マダンテ：** 全MPを消費し、消費量に応じた絶大なダメージを敵全体に与える。  
* **クロスマダンテ：** 2人のMPを全て解放する究極呪文（連携技などでも登場）。  
* **メドローア：** 炎と氷の相反する魔力を衝突させ、敵を消滅または極大ダメージを与える。  
* **天国への階段：** 神の裁きで敵1グループにダメージ。

### 2\. 回復・蘇生呪文

* **ホイミ / ベホイミ / ベホイム / ベホマ：** 味方単体のHPを小〜完全回復する。  
* **ベホマラー / ベホマズン：** 味方全体のHPを中〜完全回復する。  
* **リホイミ / リベホイミ / リベホイム：** 味方1人をHPが毎ターン自動回復する状態にする。  
* **ザオ / ザオラル：** 味方1人を確率で（またはHPわずかで）蘇生する。  
* **ザオリク / ザオリーマ：** 味方1人〜全体をHP満タンで確実に蘇生する。  
* **メガザル：** 自分の命を犠牲にして、他の味方全員を完全蘇生・回復する。  
* **キアリー / キアリク / キアラル：** それぞれ毒、マヒ・睡眠、混乱を治癒する。  
* **ザメハ：** 味方全体の眠りを覚ます。  
* **シャナク / おはらい：** 呪われた装備や状態を解除する。  
* **マホリー：** 呪文封じ状態を治す。

### 3\. 補助・弱体・探索呪文

**【味方強化（バフ）】**

* **スカラ / スクルト：** 味方単体 / 全体の守備力を上げる。  
* **バイキルト / バイシオン：** 味方単体の攻撃力を2倍、または全体を少し上げる。  
* **ピオラ / ピオリム：** 味方単体 / 全体の素早さを上げる。  
* **バーハ / フバーハ：** 味方単体 / 全体のブレス耐性を上げる。  
* **マジックバリア / マホバリア：** 味方全体 / 単体の呪文耐性を上げる。  
* **マホカンタ / マホターン：** 呪文を敵に跳ね返すバリアを張る（ターン制限などの違いあり）。  
* **マホキテ：** 受けた呪文のMPを吸収して自分のものにする。  
* **マホステ：** 対象にかかる呪文を無効化する。  
* **アストロン：** 仲間全員を鋼鉄に変え、数ターン一切の攻撃を無効化するが行動も不可になる。  
* **モシャス：** 仲間1人に変身し、ステータスや呪文をコピーする。  
* **ドラゴラム：** ドラゴンに変身し、炎を吐いて攻撃する。  
* **やまびこのさとり：** 数ターン、呪文が2回連続で発動するようになる。

**【敵弱体（デバフ）・状態異常】**

* **ルカニ / ルカナン：** 敵単体 / グループの守備力を下げる。  
* **ヘナトス：** 敵単体の攻撃力を下げる。  
* **ボミエ / ボミオス：** 敵単体 / グループの素早さを下げる。  
* **マヌーサ / マヌーハ：** 敵グループを幻惑状態にする（マヌーハは味方の幻惑を解除）。  
* **マホトーン：** 敵グループの呪文を封じる。  
* **メダパニ / メダパニーマ：** 敵単体 / 全体を混乱させる。  
* **ラリホー / ラリホーマ：** 敵グループ / 全体を眠らせる。  
* **マホトラ：** 敵1体からMPを吸い取る。  
* **ディバインスペル：** 敵単体の呪文耐性を下げる。  
* **バシルーラ：** 敵1体を戦闘から除外（遠くへ飛ばす）。  
* **ニフラム：** 敵グループを光の彼方へ消し去る。  
* **マジャスティス / ギガジャティス：** 敵の有利な効果をかき消す。ギガは全効果を消し呪文も遮断。  
* **パルプンテ：** 何が起こるかわからないランダム効果。

**【移動・探索】**

* **ルーラ / リレミト：** 行ったことのある町へ移動 / ダンジョンから脱出。  
* **トヘロス / ステルス：** 弱い敵とのエンカウントを防ぐ / 敵から見つからなくなる。  
* **トラマナ：** ダメージ床を無効化。  
* **インパス / レミラーマ：** 宝箱の中身を判別する / 落ちているアイテムを光らせる。  
* **フローミ / アバカム：** 現在の階層や場所を知る / 鍵がかかった扉を開ける。  
* **ラナルータ / レムオル：** 昼夜を逆転させる / パーティの姿を透明にする。

### 4\. 武器・物理攻撃特技

**【属性を帯びた斬撃】**

* **かえん斬り / れんごく斬り / 爆炎斬り：** 火・炎属性を付与した斬撃。  
* **いなずま斬り / 雷神斬り / 雷鳴斬り / 稲妻雷光斬：** 雷属性の斬撃（作品によりイオ系やデイン系弱点に特効）。  
* **マヒャド斬り / ひょうけつ斬り / ブリザーラッシュ / 零度斬り：** 氷属性の斬撃。  
* **しんくう斬り / 風神斬り：** 風（バギ）属性の斬撃。  
* **れっぱ斬り / ばくれん斬り / 爆砕斬り：** 爆発（イオ）属性の斬撃。  
* **ホーリーエッジ / ホーリーラッシュ / 光明斬り：** 光（デイン）属性の斬撃。  
* **ダークスパイク / ダークマッシャー：** 闇属性の斬撃。  
* **すいりゅう斬り：** 物質系に強く、たまに雷耐性を下げる船守の剣技。

**【種族特効】**

* **ドラゴン斬り / ドラゴンスラッシュ：** ドラゴン系に大ダメージ。  
* **メタル斬り：** メタル系に確実に1〜2ダメージ等を与える。  
* **ゾンビ斬り / 黄泉送り：** ゾンビ系に大ダメージ。  
* **まじゅう斬り / けもの突き：** 獣・魔獣系に大ダメージ。  
* **あくま斬り：** 悪魔系に大ダメージ。  
* **ぶっしつ斬り / だいせつだん / ドラムクラッシュ：** 物質系に大ダメージ。  
* **スライムたたき / スライム斬り / スライムブロウ：** スライム系に大ダメージ。  
* **キラーブーン：** 虫系に大ダメージ。  
* **たいぼく斬：** 植物系に大ダメージ。  
* **波紋演舞：** 水系に大ダメージ。  
* **聖魔斬：** ？？？（未知）系に大ダメージ。

**【多段・連続攻撃】**

* **はやぶさ斬り / 超はやぶさ斬り：** 2回攻撃 / 4回攻撃。（※『VIII』などでは「はやぶさの剣」装備ではやぶさ斬りをすると4回攻撃になるが、『X』『XI』ではならない）  
* **つるぎのまい / さみだれけん / さみだれづき / さみだれうち：** ランダムな敵に3〜4回の連続攻撃。  
* **氷結らんげき / 豪雪氷結乱撃：** 氷属性の4回〜5回攻撃。改は耐性低下付き。  
* **ばくれつけん / 爆裂拳 / ばくれつきゃく：** ランダムな敵に怒涛の4連打〜7連打。素手専用の作品とそうでない作品がある。  
* **タイガークロー / ライガークラッシュ / ネイルスクラッチ：** ツメによる強力な2〜5回連続攻撃。  
* **双竜打ち / 極竜打ち / みだれ打ち：** ムチによる連続攻撃。極竜打ちはバフ消し効果あり。  
* **キラージャグリング / ゴッドジャグリング：** 敵全体にランダムで6〜8回の連続攻撃。

**【強力な単体攻撃】**

* **アルテマソード：** 防御力無視、または力依存の究極の単体攻撃。作品により守備力依存（『VII』）や力依存（『XI』）の違いがある。  
* **渾身斬り / 全身全霊斬り：** 全力で振り下ろす大ダメージ剣技。  
* **もろば斬り：** 絶大なダメージを与えるが、自身も反動ダメージを受ける。  
* **ミラクルソード / ゴスペルソード：** ダメージを与えつつ、自身のHP（ゴスペルはMPも）を回復する。  
* **一閃突き / 雷光一閃突き / まじん斬り / 大まじん斬り：** 命中率は低いが当たれば必ず会心の一撃（一閃突きの一部作品は例外）。  
* **アサシンアタック / ライトニングデス：** 急所を突き、確率で即死させる。  
* **タナトスハント / ヒュプノスハント：** 毒・マヒ状態、または眠り・混乱状態の敵に通常の6倍などの絶大ダメージ。  
* **せいけん突き：** 岩石耐性（作品により異なる）に依存して大ダメージを与える拳技。  
* **ゴールドフィンガー：** ダメージを与えつつ敵のバフ（有利な状態）を消し去る。  
* **ゴッドスマッシュ：** 神の閃光とともに打ち抜く光属性の単体大ダメージ。

**【全体・範囲攻撃】**

* **ギガスラッシュ / ギガブレイク / ギガクラッシュ：** 稲妻を宿した剣で敵全体またはグループに大ダメージ。  
* **ぶんまわし / 大旋風斬り / なぎ払い / 断空なぎ払い：** 武器を振り回しグループや周囲の敵を攻撃。  
* **オノむそう / 真・オノむそう：** 斧を振り回しグループや全体を攻撃。  
* **ランドインパクト / プレートインパクト / ビッグバン：** 地面を叩き割ったり大爆発を起こし全体攻撃。  
* **クロスカッター / パワフルスロー / 超パワフルスロー：** ブーメランで敵全体を攻撃（減衰しない技もある）。  
* **シャインスコール / バーニングバード / フローズンバード：** 光の雨や炎/氷の鳥を放ち全体やランダムに攻撃。  
* **岩石おとし / しんくうは：** 巨大な岩石や真空の刃で敵全体を攻撃。  
* **ムーンサルト / ミラクルムーン：** 空中から全体を攻撃。敵の数で威力が変動（ミラクルは回復付き）。

**【状態異常・デバフ付与打撃】**

* **ポイズンダガー / スリープダガー / しばり打ち / らせん打ち：** 攻撃と同時に毒、眠り、マヒ、混乱を付与。  
* **かぶとわり / 鉄甲斬 / やいばくだき：** 攻撃と同時に守備力や攻撃力を下げる。  
* **ハートブレイク / マインドブレイク：** 攻撃しつつ確率で1ターン行動不能（休み）にする。  
* **ぬすっと斬り / ぬすっと刈り / 大どろぼうの鎌：** 攻撃と同時にアイテムを盗む。

**【特殊物理・その他】**

* **しっぷう突き / しっぷう攻撃：** ターンの最初に先制攻撃するが威力は弱め。  
* **におう斬り：** 敵の「みがわり」「におうだち」を無視して攻撃する。  
* **天下無双：** 敵単体に怒涛の6回連続攻撃。  
* **シールドアタック：** 防御しながら攻撃。  
* **ゴールドハンマー / ゴールドラッシュ：** 攻撃しつつゴールドを奪う。

### 5\. ブレス（息）・自然系特技

**【ブレス（息）系】**

* **炎：** 火の息、火炎の息、激しい炎、しゃくねつ、煉獄火炎、地獄の業火など。  
* **氷：** 冷たい息、凍りつく息、凍える吹雪、輝く息、絶対零度など。  
* **状態異常・妨害：** あまい息（眠り）、どくの息、もうどくのきり（猛毒）、やけつく息（マヒ）、まどいの息（混乱）、斬撃/体技/作戦封じの息、死神の息吹（即死）、レクイエムのふえ、大きなため息（テンション低下）。  
* **特殊属性：** やみのほのお、ダークネスブレス、ひかりのブレス、シャイニングブレス、オーロラブレスなど。

**【自然・体術系】**

* **風・水・雷属性：** かまいたち、トルネード、地獄の竜巻、いなずま、はげしいいなずま、てっぽうみず、つなみ、巨大なつなみ。  
* **大地・岩石属性：** じひびき、ネクロゴンドの波動、じわれ、めいどうふうま（地割れで即死/ダメージ）、てんぺんちい、大地揺らし、ガレキ落とし。  
* **宇宙・その他：** 星くず、流星、ブラックホール、神々の怒り。

### 6\. 歌・踊り系特技

**【踊り】**

* **攻撃・妨害：** 死のおどり（全体即死）、マッスルダンス（物理）、メダパニダンス（混乱）、さそうおどり / ステテコダンス（休み）、おどりふうじ、しらけるダンス（テンション低下）。  
* **味方支援：** ハッスルダンス（全体回復）、メガザルダンス（自己犠牲蘇生）、たたかいのまい（攻撃力上昇）、アゲアゲダンス。  
* **特殊：** バックダンサー呼び（全体大ダメージ）、地獄のおどり（MP全消費ランダム攻撃）。

**【歌】**

* **攻撃・妨害：** 死の曲（全体即死）、ひつじかぞえ歌 / ゆりかごの歌（眠り）、のろいの歌（守備力低下）、まふうじの歌。  
* **味方支援：** やすらぎの歌（全体回復）、天使の歌声 / せいれいのうた（蘇生）、たたかいのふえ / まもりのふえ（バフ付与）、覚醒のアリア。
