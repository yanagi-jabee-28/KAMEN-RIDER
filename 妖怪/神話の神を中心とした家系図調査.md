### **序論**

この報告書は、8世紀に編纂された日本の二大古典である『古事記』と『日本書紀』に記録されている神々の系譜を包括的に分析するものである。両書はそれぞれ異なる物語的・政治的目的を持つため、神々の系譜に顕著な相違が見られる。これらの系譜は単なる家族関係の記録ではなく、ヤマト王権の皇統の神聖な主権を正当化し、有力氏族の階級的地位を確立するために設計された、根源的な政治憲章としての役割を担っている。本報告書では、原初の混沌から初代天皇の祖先へと至る神々の系譜を体系的に追跡し、文献間の差異とその学術的解釈に焦点を当てる。

### **第1部 万神殿の創生：混沌から創造へ**

この章では、宇宙の創生と神々の誕生について詳述し、神話全体の基盤を確立する。抽象的な神々から人格神への変遷を追う。

**1.1 原初の神々：別天津神（ことあまつかみ）と神世七代（かみよななよ）**
*   **天地開闢と三柱の神**：世界の最初に高天原（たかまのはら）に天之御中主神（アメノミナカヌシノカミ）、高御産巣日神（タカミムスヒノカミ）、神産巣日神（カミムスヒノカミ）の三柱が出現した。これらは独神（ひとりがみ）で、すぐに姿を隠したとされ、宇宙を始動させた抽象的な力として描かれる。
*   **神世七代**：続いて、男女一対の神々の概念を導入する七世代の神々「神世七代」が現れる。これにより、創造の形式が抽象的なものから生殖によるものへと移行する。
*   **イザナギとイザナミ**：神世七代の最後に現れたのがイザナギノミコトとイザナミノミコトであり、彼らは他の神々から「この漂える国を修理（つく）り固め成せ」という指令を受けた。

**1.2 神聖なる夫婦：イザナギノミコトとイザナミノミコト**
*   **淤能碁呂島（おのごろじま）の創造**：天浮橋（あめのうきはし）から天沼矛（あめのぬぼこ）で混沌をかき混ぜ、最初の島である淤能碁呂島を創り出した。
*   **求婚の儀式と失敗**：天の御柱（あめのみはしら）を巡る求婚の儀式で、女神であるイザナミが先に声をかけたため、不具の子である水蛭子（ヒルコ）が生まれた。これは、女性が先に口を開くことの不適切さを強調し、家父長制的な秩序を確立する物語上の重要な点とされる。

**1.3 国生みと神生み**
*   **国生み**：儀式を正しくやり直した後、二神は日本の国土創成（国生み）を開始。淡路島から始まり、四国、九州、本州など八つの主要な島々（大八島国）を生んだ。『古事記』と『日本書紀』では、島々が生まれる順序に若干の差異が見られる。
*   **神生み**：国土創成後、海、風、山、木々など自然現象を神格化した多様な神々を生み出した（神生み）。この一連の創造は、日本の国土とその環境を神聖化し、物理的な風景の中に神性を埋め込む行為でもある。

**1.4 カグツチの悲劇と黄泉国（よみのくに）への旅**
*   **イザナミの死**：火の神、火之迦具土神（ヒノカグツチノカミ）を産んだ際、イザナミはその炎で女陰を焼かれて命を落とし、世界に「死」がもたらされた。
*   **タケミカヅチの誕生**：悲しんだイザナギはカグツチを十拳剣（とつかのつるぎ）で斬り殺し、その剣に付着した血から雷と剣の神である建御雷之男神（タケミカヅチノオノカミ）などが誕生した。タケミカヅチは後の国譲り神話で重要な役割を果たす。
*   **黄泉国**：イザナギは妻を取り戻すため死者の国「黄泉国」へ旅立つが、禁忌を破って腐敗したイザナミの姿を見てしまい、二人は永遠に離別する。この神話は生と死の境界を確立し、生命の循環の起源を説明している。

### **第2部 三貴子：天と地の領域を支える柱**

この章では、アマテラス、ツクヨミ、スサノオの「三柱の貴き子（みはしらのうずのみこ）」に焦点を当てる。彼らは天、夜、海を支配し、宇宙の基本構造を確立する。

**2.1 禊（みそぎ）からの誕生**
黄泉国の穢れから逃れたイザナギが川で禊を行った際に三貴子が生まれた。
*   **天照大御神（アマテラスオオミカミ）**：左目を洗ったときに生まれた太陽の女神。
*   **月読命（ツクヨミノミコト）**：右目を洗ったときに生まれた月の神。
*   **建速須佐之男命（タケハヤスサノオノミコト）**：鼻を洗ったときに生まれた嵐の神。

イザナギは彼らに高天原、夜の食国（よるのおすくに）、海原の統治をそれぞれ命じた。この誕生経緯は、神話のテーマが生殖による創造から浄化による創造へ移行したことを示し、「祓い」という神道の価値観を強化している。

**2.2 天照大御神：太陽の女神と皇室の祖先**
アマテラスは高天原の主神であり、日本の皇室の祖先とされる。彼女を祀る伊勢神宮は神道で最も神聖な場所である。弟スサノオの乱暴により天岩戸（あまのいわと）に隠れ、世界が闇に包まれる神話は、秩序と光の回復を象徴する物語である。

**2.3 月読命：謎に包まれた月の神**
*   **『日本書紀』の神話**：食物の神である保食神（ウケモチノカミ）が口から食物を出して饗応したことに激怒し、彼女を斬り殺す。ウケモチの亡骸からは牛馬や稲、蚕などが生じた。この凶行に怒ったアマテラスが「もはや会いたくない」と宣言したことが、昼と夜が分かれた起源とされる。
*   **『古事記』との相違点**：『古事記』では、ほぼ同一の筋書きがスサノオの行為として語られ、食物の女神の名も大宜都比売神（オオゲツヒメノカミ）となっている。この差異は、神々の役割の流動性を示唆している。

**2.4 須佐之男命：荒ぶる神と出雲王朝の創始者**
スサノオは海原の統治を拒み、母の国へ行きたいと泣き叫んだため、高天原で乱暴を働き、天界から追放される。
*   **英雄への変貌**：地上に降りた後、出雲で八岐大蛇（やまたのおろち）を退治し、櫛名田比売（クシナダヒメ）を救う文化英雄となる。大蛇の尾から三種の神器の一つである草薙剣（くさなぎのつるぎ）を発見し、アマテラスに献上して和解した。
*   **出雲の祖神**：クシナダヒメと結婚して出雲に定住し、大国主神（オオクニヌシノカミ）に代表される国津神（地上の神々）の系譜の祖となった。彼の二面性は、天津神と国津神の橋渡し役を担っている。

### **第3部 天津神の系譜：皇統治世への道**

この章では、アマテラスの直系子孫をたどり、皇室の正統性の神話的基盤を詳述する。

**3.1 神々の誓約（うけい）：アマテラスとスサノオの子孫**
スサノオが高天原を去る前、二神は互いの持ち物を交換して神々を生み出す「誓約」を行った。
*   アマテラスはスサノオの剣から三柱の女神（宗像三女神）を生んだ。
*   スサノオはアマテラスの勾玉から五柱の男神（アメノオシホミミなど）を生んだ。
アマテラスは五柱の男神が自身の持ち物から生まれたため自分の子であると宣言し、長男のアメノオシホミミを後継者とすることで、正統な継承系統を確立した。

**3.2 天孫降臨：ニニギノミコトの系譜**
アマテラスは、自身の孫である**瓊瓊杵尊（ニニギノミコト）**に三種の神器を授け、地上を統治するよう命じた。ニニギは日向（九州）の高千穂の峰に降臨する。この「天孫降臨」は、天上の権威と地上の主権を直接結びつけ、皇室の神聖な系譜を確立する中心的な物語である。
*   **サルタヒコとの出会い**：降臨の途中、国津神の猿田毘古神（サルタヒコノカミ）と出会い、女神アメノウズメノミコトの仲介を経て、サルタヒコが天孫を導く。これは天と地の権威の平和的統合を象徴する。
*   **コノハナノサクヤヒメとの結婚**：ニニギは山の神の娘である美しい木花之佐久夜毘売（コノハナノサクヤヒメ）と結婚し、容姿の劣る姉の石長比売（イワナガヒメ）を退けた。この選択が、天皇ひいては人類が短命であることの神話的起源とされる。

**3.2.1 ニニギノミコトの家系図（簡潔版）**
*   **系譜**：アマテラス → アメノオシホミミ → ニニギノミコト
*   **配偶者**：木花之佐久夜毘売（コノハナノサクヤヒメ）
*   **子**：ホデリ（海幸彦）、ホオリ（山幸彦）、ホスセリ
*   **天孫から初代天皇まで**：ニニギ → ホオリ → 鵜葺草葺不合命（うがやふきあえず） → 神倭伊波礼毘古命（神武天皇）
*   **ポイント**：ニニギは天孫として皇位継承の神話的根拠となり、妻との逸話は地域的繋がりや「短命性」の語源説明として機能する。

**3.3 五伴緒（いつのともお）と有力氏族の起源**
ニニギは、ヤマト王権の祭祀や軍事を司る有力氏族の祖先となる五柱の神々「五伴緒」を伴って降臨した。

| 神祖 | 氏族 | 世襲的役割・職能 |
| :--- | :--- | :--- |
| **天児屋命** | **中臣氏** (後の藤原氏) | 宮廷祭祀、祝詞 |
| **布刀玉命** | **忌部氏** | 神聖な供物、神殿造営 |
| **天宇受売命** | **猿女君** | 神楽、宮廷の舞 |
| **伊斯許理度売命** | **鏡作連** | 鏡の製作 |
| **玉祖命** | **玉祖連** | 玉の製作 |

**3.4 日向三代：ニニギから初代天皇の父へ**
九州の日向に居住した三世代の神々。
1.  **第一世代：ニニギノミコト**。子にホデリ（海幸彦）と**ホオリ**（山幸彦）がいる。
2.  **第二世代：ホオリノミコト**。兄の釣り針をなくした神話がきっかけで、海の神ワタツミの娘である**豊玉毘売（トヨタマヒメ）**と結婚する。
3.  **第三世代：ウガヤフキアエズノミコト**。トヨタマヒメが出産中に正体（ワニ）を見られ、海に去る。子は妹の**玉依毘売（タマヨリヒメ）**に託され、後に叔母である彼女と結婚する。
4.  **初代天皇の祖**：ウガヤフキアエズとタマヨリヒメの子の一人が、初代天皇となる**神武天皇**である。

**3.5 大山津見神（オオヤマツミ）と大綿津見（ワタツミ）の位置付け**
*   **大山津見神（山の神）**：ニニギの妻コノハナノサクヤヒメとイワナガヒメの父。天孫と山地の在地勢力を結びつける。
*   **大綿津見（海の神）**：ホオリの妻トヨタマヒメの父。皇統に海の系譜を組み込む役割を果たす。
両神の娘たちが天孫の子孫と婚姻することで、天孫系と各地域勢力の神聖な結合が神話的に説明されている。

### **第4部 国津神の系譜：出雲の地の神々**

この章では、天津神と並行して展開される地上の神々（国津神）の系譜、特にオオクニヌシと「国譲り」の神話に焦点を当てる。

**4.1 国土の偉大なる主：オオクニヌシノカミの系譜と試練**
オオクニヌシとスサノオの関係には、文献間で顕著な矛盾が見られる。

| 文献 | 記述された関係 | 示唆される意味 |
| :--- | :--- | :--- |
| **『古事記』** | 六世の孫 | 出雲王朝の長く独立した歴史を強調する。 |
| **『日本書紀』（本文）** | 子 | 出雲の歴史を短縮し、天孫族との近親関係を構築する。 |
| **『日本書紀』（一書）**| 五世または六世の孫 | 『古事記』の伝承に近い記述。 |

この矛盾は、ヤマト中心の史観を強化するための意図的な政治戦略を反映している。オオクニヌシは兄弟に迫害された後、祖先スサノオから試練を乗り越え、「大国主神」の名を得て地上世界の支配者となる。

**4.2 オオクニヌシの配偶者と数多の子孫**
オオクニヌシは各地で多くの妻を娶り、多くの子をもうけた。これは彼の「国造り」を象徴する。
*   **スセリビメ**：正妻。
*   **タキリビメ**：子にアヂスキタカヒコネがいる。
*   **カムヤタテヒメノミコト**：子の**事代主神（コトシロヌシノカミ）**は国譲りで重要な役割を果たす。
*   **ヌナカワヒメ**：子の**建御名方神（タケミナカタノカミ）**も国譲りで重要な役割を担う。

**4.3 国譲り：タケミカヅチとの対決**
高天原の神々は、オオクニヌシが平定した地は天孫が治めるべきと宣言し、使者として雷神**タケミカヅチノカミ**を派遣する。
*   オオクニヌシは判断を息子たちに委ね、コトシロヌシは平和的に同意するが、タケミナカタはタケミカヅチに力比べを挑み（相撲の起源）、敗れて降伏する。
*   オオクニヌシは、自身を祀る壮大な宮殿（出雲大社の起源）の建立と引き換えに国を譲り、目に見えない神事を司る存在へと退いた。これは、天皇の政治的統治と出雲の霊的権威という二元的な権力構造の前例となる。

### **第5部 主要な神々と傍系の系譜**

この章では、その他の重要な神々の系譜と神話を分析する。

**5.1 長子：ヒルコとアワシマの運命**
イザナギとイザナミの最初の御子ヒルコは、儀式の誤りのために不具の子として生まれ、葦の舟で海に流された。しかし、後に七福神の一柱である「えびす神」と習合され、漁師や商人の守護神として再生した。ただし、えびす神はコトシロヌシと同一視されることもある。

**5.2 食物の神々：ウカノミタマとオオゲツヒメの矛盾する系譜**
穀物の神、宇迦之御魂神（ウカノミタマノカミ）の出自には文献間で対立がある。

| 神 | 主要な神格 | 『古事記』における親神 | 『日本書紀』における親神 |
| :--- | :--- | :--- | :--- |
| **宇迦之御魂神** | 穀物、食物 | スサノオと神大市比売 | イザナギとイザナミ |
| **大国主神** | 国土の主、医療 | スサノオ（六世の孫として） | スサノオ（子として） |
| **建御雷神** | 雷、剣術 | カグツチを斬った剣の血 | （同様の記述） |

この系譜の対立は、農業の象徴性を巡る出雲系とヤマト系の間の緊張関係を反映している。また、食物の女神オオゲツヒメは『古事記』でスサノオに、『日本書紀』で対応するウケモチはツクヨミに殺害されるという類似の神話がある。

**5.3 導き手と使者：サルタヒコとアメノウズメの関係**
サルタヒコは天孫降臨を導く国津神、アメノウズメは天岩戸神話で活躍する天津神である。サルタヒコはアメノウズメによって伊勢まで送られ、後の伝承では夫婦と見なされるようになった。この結合は、天津神と国津神の調和のとれた結婚を象徴しており、アメノウズメは彼の名を負って「猿女君（さるめのきみ）」として知られるようになった。

### **結論：根源神話としての神統譜**

日本神話の神々の系譜は、単なる宗教物語ではなく、緻密に構築された根源神話であり、以下の三つの主要な機能を果たしている。

1.  **皇室主権の正当化**：アマテラスから天皇に至る断絶のない神聖な血統を確立し、統治の正当性を提供する。
2.  **社会的・政治的階層の法典化**：有力氏族に神聖な起源を与え、世襲的役割を正当化する。
3.  **国家的物語の創造**：日本の国土を神聖化し、民や慣習の起源を説明し、多様な地域の神々をヤマト中心の万神殿に統合する。

『古事記』と『日本書紀』に見られる矛盾や差異は、欠陥ではなく、自らの神聖な起源を定義しようとした8世紀日本のイデオロギー的関心と政治的野心を反映する貴重な洞察を提供するものである。
